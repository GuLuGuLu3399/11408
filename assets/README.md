# assets/

公开资源目录（图片 / PDF / 附件），供站点与构建时引用。本目录内容**会进入公开仓库**。

## 约定路径（从仓库根出发）

- 图片：`/assets/images/<name>.png`
- 文件：`/assets/files/<name>.pdf`

## 子目录

- `images/` — png / jpg / webp / svg / gif
- `files/` — pdf / zip / docx 等附件

> 注意：大型原始资料、拆分 PDF、RAG 派生文档、向量库与模型缓存不放在本 vault 中。它们位于旁路工作区 `../11408-kb/`，避免 Tolaria 扫描机器生成的 Markdown。
