## Q1 · 隐函数求导写错 → 面积 $1/2$ 而非 $e/2$ 🔴

### 你的错因
隐函数求导时 $y'(0)$ 算错 → 切线斜率错 → 三角形面积错。

### 正解
$e^y+xe^yy'+3x^2=0$ 代入 $(0,1)$：$e+0+0=0$？不对——重新审视，原方程对 $x$ 求导后代入。

### Fix Plan
**隐函数求导后回代验证**：求完 $y'(0)$ → 代回原方程的导数式检查是否恒等。1 分钟验算避免整题崩。

## Q4 · 微分方程忘解法 + 算错 🔴

### 你的错因
忘 $y''+3y'+2y=e^{-x}$ 怎么解 → 回去复习 → 果断算错。

### 正解
特征方程 $r^2+3r+2=0$ → $r=-1,-2$；特解 $y^*=axe^{-x}$（$e^{-x}$ 与特征根重合 → 乘 $x$）。

### Fix Plan
**ODE 解法速查卡**（背熟）：
- 齐次 $r^2+pr+q=0$ → 特征根
- $f(x)=e^{\lambda x}P(x)$ → 特解 $x^k Q(x)e^{\lambda x}$（$k$ = $\lambda$ 与特征根重合次数）

## Q5 · 绕 $y=x$ 旋转——反函数对称 + 通用公式 🔴

### 你的错因
"瞎蒙的"——没想到 $\sqrt{x}$ 与 $x^2$ 互为反函数，关于 $y=x$ 对称 → **只需算一条曲线**。公式也记错了。

### 正解（RAG 提取）
$\sqrt{x}$ 与 $x^2$（$x\geq0$）互为反函数 → 图像关于 $y=x$ 对称 → 绕 $y=x$ 旋转只算一条。

**绕任意直线 $Ax+By+C=0$ 的通用体积公式**（30 讲结论）：
$$V=\frac{\pi}{(A^2+B^2)^{3/2}}\int_a^b[Ax+Bf(x)+C]^2|Af'(x)-B|\,dx$$

本题 $y=x$ 即 $A=1,B=-1,C=0$：
$$V=\frac{\pi}{2\sqrt2}\int_0^1(x-x^2)^2(2x+1)\,dx=\frac{\sqrt2}{60}\pi$$

### Core Concept
**反函数对称 → 绕 $y=x$ 旋转只需算一条曲线**。这是几何应用的高频考点——见到两条曲线关于 $y=x$ 对称 → 直接砍半。

## Q6 · 壳法半径写 $x$（负区间）→ 算出负值 🔴

### 你的错因
积分区间含负数（$(-\infty,0]$），壳法半径写成了 $x$（负值），导致体积为负 → 怀疑自己。

### 正解（用户补图提取）
$D$ 绕 $x=1$ 旋转。圆盘法（不是壳法）：$\mathrm{d}V=\pi[(1-\ln y)^2-(1-y/e)^2]\,dy$，体积 $V=\frac53\pi e$。

### Error Pattern
**壳法/圆盘法的半径必须是距离（≥ 0）**。如果积分变量在负区间，半径要取 $|x|$ 或换元到正区间。

| 方法 | 半径 | 本题适用？ |
|---|---|---|
| 圆盘法（绕 $x=1$） | $1-\ln y$（外），$1-y/e$（内） | ✅ 官方解法 |
| 壳法（绕 $x=1$） | $|1-x|$（到旋转轴的距离） | 可以但更绕 |

### Fix Plan
**写壳法/圆盘法之前先问自己：半径的表达式 ≥ 0 吗？** 如果 $\leq 0$ → 加绝对值或换圆盘法。

## Q7 · 漏分段（$\sin x=0$ 瑕点）→ 直接积分出错 🔴

### 你的错因
$y=e^{-x/2}\sqrt{\sin x}$ → 被 $\sqrt{\sin x}$ 蒙了，没意识到 $\sin x=0$ 的点是瑕点 → 直接积分 → 错。

### 正解（RAG 提取）
$\sin x\geq0$ 的区间是 $[2k\pi,(2k+1)\pi]$ → **分段积分**，每段：
$$V_k=\pi\int_{2k\pi}^{(2k+1)\pi}e^{-x}\sin x\,dx=\frac12\pi(1+e^{-\pi})e^{-2k\pi}$$
无穷多个 $V_k$ 构成等比数列 → 求和：
$$V=\sum_{k=0}^{\infty}\frac12\pi(1+e^{-\pi})e^{-2k\pi}=\frac{\pi}{2(1-e^{-\pi})}$$

### Core Concept
**$\sqrt{\sin x}$ → 先找 $\sin x=0$ 的点 → 必分段**。无穷多段 + 每段值构成等比 → 等比求和。

### 识别信号
被积函数含 $\sqrt{\sin/\cos}\cdot$ → **先查零点分段**（$\sin x=0$ at $k\pi$；$\cos x=0$ at $\frac\pi2+k\pi$）。
