## Q1 · 隐函数求导写错 → 面积 $1/2$ 而非 $e/2$ 🔴

### 你的错因
隐函数求导时 $y'(0)$ 算错 → 切线斜率错 → 三角形面积错。

### 正解
$e^y+xe^yy'+3x^2=0$ 代入 $(0,1)$：$e+0+0=0$？不对——重新审视，原方程对 $x$ 求导后代入。

### Fix Plan
**隐函数求导后回代验证**：求完 $y'(0)$ → 代回原方程的导数式检查是否恒等。1 分钟验算避免整题崩。

## Q4 · 微分方程忘解法 + 算错 🔴

### 你的错因
忘 $y''+3y'+2y=e^{-x}$ 怎么解 → 回去复习 → 果断算错。

### 正解
特征方程 $r^2+3r+2=0$ → $r=-1,-2$；特解 $y^*=axe^{-x}$（$e^{-x}$ 与特征根重合 → 乘 $x$）。

### Fix Plan
**ODE 解法速查卡**（背熟）：
- 齐次 $r^2+pr+q=0$ → 特征根
- $f(x)=e^{\lambda x}P(x)$ → 特解 $x^k Q(x)e^{\lambda x}$（$k$ = $\lambda$ 与特征根重合次数）

## Q5 · 绕 $y=x$ 旋转——反函数对称 + 通用公式 🔴

### 你的错因
"瞎蒙的"——没想到 $\sqrt{x}$ 与 $x^2$ 互为反函数，关于 $y=x$ 对称 → **只需算一条曲线**。公式也记错了。

### 正解（RAG 提取）
$\sqrt{x}$ 与 $x^2$（$x\geq0$）互为反函数 → 图像关于 $y=x$ 对称 → 绕 $y=x$ 旋转只算一条。

**绕任意直线 $Ax+By+C=0$ 的通用体积公式**（30 讲结论）：
$$V=\frac{\pi}{(A^2+B^2)^{3/2}}\int_a^b[Ax+Bf(x)+C]^2|Af'(x)-B|\,dx$$

本题 $y=x$ 即 $A=1,B=-1,C=0$：
$$V=\frac{\pi}{2\sqrt2}\int_0^1(x-x^2)^2(2x+1)\,dx=\frac{\sqrt2}{60}\pi$$

### Core Concept
**反函数对称 → 绕 $y=x$ 旋转只需算一条曲线**。这是几何应用的高频考点——见到两条曲线关于 $y=x$ 对称 → 直接砍半。

## Q6 · 壳法半径写 $x$（负区间）→ 算出负值 🔴

### 你的错因
积分区间含负数（$(-\infty,0]$），壳法半径写成了 $x$（负值），导致体积为负 → 怀疑自己。

### 正解（用户补图提取）
$D$ 绕 $x=1$ 旋转。圆盘法（不是壳法）：$\mathrm{d}V=\pi[(1-\ln y)^2-(1-y/e)^2]\,dy$，体积 $V=\frac53\pi e$。

### Error Pattern
**壳法/圆盘法的半径必须是距离（≥ 0）**。如果积分变量在负区间，半径要取 $|x|$ 或换元到正区间。

| 方法 | 半径 | 本题适用？ |
|---|---|---|
| 圆盘法（绕 $x=1$） | $1-\ln y$（外），$1-y/e$（内） | ✅ 官方解法 |
| 壳法（绕 $x=1$） | $|1-x|$（到旋转轴的距离） | 可以但更绕 |

### Fix Plan
**写壳法/圆盘法之前先问自己：半径的表达式 ≥ 0 吗？** 如果 $\leq 0$ → 加绝对值或换圆盘法。

## Q7 · 漏分段（$\sin x=0$ 瑕点）→ 直接积分出错 🔴

### 你的错因
$y=e^{-x/2}\sqrt{\sin x}$ → 被 $\sqrt{\sin x}$ 蒙了，没意识到 $\sin x=0$ 的点是瑕点 → 直接积分 → 错。

### 正解（RAG 提取）
$\sin x\geq0$ 的区间是 $[2k\pi,(2k+1)\pi]$ → **分段积分**，每段：
$$V_k=\pi\int_{2k\pi}^{(2k+1)\pi}e^{-x}\sin x\,dx=\frac12\pi(1+e^{-\pi})e^{-2k\pi}$$
无穷多个 $V_k$ 构成等比数列 → 求和：
$$V=\sum_{k=0}^{\infty}\frac12\pi(1+e^{-\pi})e^{-2k\pi}=\frac{\pi}{2(1-e^{-\pi})}$$

### Core Concept
**$\sqrt{\sin x}$ → 先找 $\sin x=0$ 的点 → 必分段**。无穷多段 + 每段值构成等比 → 等比求和。

### 识别信号
被积函数含 $\sqrt{\sin/\cos}\cdot$ → **先查零点分段**（$\sin x=0$ at $k\pi$；$\cos x=0$ at $\frac\pi2+k\pi$）。

## Q5（第 11 章）· 积分不等式——构造 $G(x)$ 辅助函数 🔴

### Question
已知 $\int_a^x f(t)dt \geq \int_a^x g(t)dt$（$\forall x\in[a,b]$）且 $\int_a^b f = \int_a^b g$，证明 $\int_a^b xf(x)dx \leq \int_a^b xg(x)dx$。

### My Answer
分部积分凑出 $xf(x)$、$xg(x)$，但**无法比较大小**。

### Correct Answer（RAG 提取）

**① 构造辅助函数** $F(x)=f(x)-g(x)$，$G(x)=\int_a^x F(t)\,dt$。
- 由题意 $G(x) \geq 0$，$G(a)=0$，$G(b)=0$。
- 注意 $G'(x)=F(x)$。

**② 分部积分** $\int_a^b xF(x)\,dx = \int_a^b x\,d[G(x)] = xG(x)\big|_a^b - \int_a^b G(x)\,dx$。
- 边界项 $= bG(b) - aG(a) = 0$。

**③ 定号** $= -\int_a^b G(x)\,dx \leq 0$（因为 $G \geq 0$）。

### Error Pattern
你分部积分做对了，但**直接比较 $f$ 和 $g$**——而题目给的是**积分关系**（$\int f \geq \int g$），不是函数关系。**积分不等式的核心是构造 $G = \int F$，把积分关系转化为 $G$ 的符号判断。**

| 你卡在哪 | 正解突破 |
|---|---|
| 比较 $xf(x)$ vs $xg(x)$ | **不比较原函数**，构造 $G(x)=\int_a^x[f-g]$ |
| 分部积分后不知道怎么办 | 分部积分的目标是**把 $xF$ 转化为 $G$ 的积分** |
| 不知道 $G \geq 0$ | 由题意 $\int_a^x f \geq \int_a^x g$ → $G(x) \geq 0$ |

### 识别信号
积分不等式 + 已知 $\int f \geq \int g$ → **构造 $G(x) = \int_a^x [f-g]dt$** → 分部积分消边界 → 用 $G \geq 0$ 定号。

## 18 讲第 11 讲总结——积分等式与积分不等式（RAG 提取）

### 三向解题法框架

| 部分 | 武器 | D 编号 |
|---|---|---|
| **等式** | 与字母无关性、线性性、方向性 | $D_1$ |
| | **祖孙三代** $(\int_a^x f, f, f')$ 奇偶性 | $D_1$ |
| | 祖孙三代**周期性** | $D_1$ |
| | 华里士公式 | $D_1+D_{23}$ |
| | 积分中值定理 | $D_1$ |
| | 换元积分法 / 区间再现 | $D_1+D_{23}$ |
| | 牛顿-莱布尼茨公式 | $D_1+D_{23}$ |
| | 分部积分法 | $D_1+D_{23}+D_{43}$(数形结合) |
| | 可加(可拆)性 | $D_1+D_{23}$ |
| **不等式** | 比较定理(保号性) | $D_1$ |
| | 构造辅助函数 | $D_1+D_{22}$+D_{23}$ |

### 黎曼思想——积分不等式的灵魂

> **黎曼思想的核心**：定积分是**面积的累积**，不是"一个公式算出来一个数"。

积分不等式的本质：**利用 $G(x) = \int_a^x F(t)\,dt$ 的"累积方向"来判断符号。**

| 已知 | 累积效应 | 结论 |
|---|---|---|
| $F(t) \geq 0$ | $G(x)$ 单调增 | $G(x) \geq G(a) = 0$（面积越积越多） |
| $F(t) \leq 0$ | $G(x)$ 单调减 | $G(x) \leq G(a) = 0$ |
| $\int_a^x f \geq \int_a^x g$ | $G(x) = \int_a^x[f-g] \geq 0$ | $G$ 的面积恒非负 |

**这就是"黎曼思想"在积分不等式中的体现**：

```
函数关系 → 积分关系 → 符号判断
f ≥ g    → G ≥ 0    → ∫xF = -∫G ≤ 0
（点态）   （累积）    （分部积分后定号）
```

**你 Q5 卡住的根因**：你在"函数关系"层面找比较（$xf$ vs $xg$），但题目给的是"积分关系"（$\int f \geq \int g$）。黎曼思想告诉你：**不要比较函数本身，比较它们的积分（累积量），用累积量的符号来定方向。**

### 祖孙三代（等式问题的高频武器）

$(\Phi(x), f(x), f'(x))$ 其中 $\Phi(x) = \int_a^x f(t)\,dt$。

| 性质 | 内容 | 常考点 |
|---|---|---|
| **奇偶性传递** | $f$ 奇 → $\Phi$ 偶；$f$ 偶 → $\Phi$ 不一定 | 对称区间积分归零/翻倍 |
| **周期性传递** | $f$ 周期 $T$ → $\int_a^{a+nT} f = n\int_0^T f$ | 无穷多周期积分 = 有限值 $\times n$ |

### Fix Plan（积分不等式标准三步）

```
① 构造 G(x) = ∫[差函数] dt    → 把积分关系转化为 G 的符号
② 分部积分                    → 把 xF 转化为 G 的表达式
③ 用 G ≥ 0 (或 ≤ 0) 定号      → 黎曼累积方向决定不等式方向
```
