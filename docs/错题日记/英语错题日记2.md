# 错题日记 11 · 英语

> Day 2 英语：英语二 2014 Text 2（自我评价偏高）Q3 + 完型填空（肥胖与健康）8 错 + 100 单词。

## 阅读 Q3 · 过度引申 🔴

### Question
英语二 2014 Text 2「人们自我评价偏高」（self-enhancement / superiority illusion）。Q3 关于 visual recognition 的性质。

### My Answer
**选了 C**。以为"make up"是人们为了内心自信**主动编造**、**掩盖沮丧**。

### Correct Answer
**A**（our self-ratings are unrealistically high）。

### 错因诊断

| 你的理解 | 原文实际 |
|---|---|
| **有意识**地 make up（编造）→ 掩盖沮丧 → 提升自信 | **无意识 / 自动**的心理过程（automatic psychological process）→ 高估自己 → 根深蒂固的本能 |
| 防御机制（你加的因果链） | 原文没提"沮丧"也没提"掩盖" |

> **错误类型：过度引申**——读懂了原文但加了自己的推理。原文只说"automatic process → overestimate"，你加了一步"make up → cover depression"。**正确选项一定能从原文原句找到同义替换，不需要额外推理。**

### Fix Plan
每个选项问自己：**"原文哪句话能直接支撑？"** 如果需要你自己加一步"中间推理"→ 排除。

### 识别信号
题干涉及心理学概念（self-enhancement / illusion / bias）→ **忠于原文用词**，别用自己的心理学知识补脑。

## 完型填空 · 8 错（首段定调没接住）🔴

### Question
英语二 2014 完型「Thinner isn't always better」（瘦不一定好）。错 1/2/3/5/7/8/18/19。

### 错题对照

| 题 | 正确答案 | 词 | 错因 |
|---|---|---|---|
| 1 | **concluded** | denied/doubled/ensured | 研究**得出结论**，不是 ensure（确保） |
| 2 | **protective** | dangerous/sufficient/troublesome | 肥胖对某些病有**保护**作用——反直觉 |
| 3 | **However** | Instead/Likewise/Therefore | 转折：瘦人不一定好 |
| 5 | **well-being** | welfare/well-off/wealth | 健康**福祉**固定搭配 |
| 7 | **workouts** | issues/ages/anxieties | 需要**锻炼** |
| 8 | **functions** | forms/sizes/categories | 大脑**功能**提升 |
| 18 | **impact** | effect/result/outcome | have an **impact** on 固定搭配 |
| 19 | **compared** | known/defined/associated | **compared** with 固定搭配 |

### 核心错因：首段定调没接住

首句 **"Thinner isn't always better"** = 全文主题句。逻辑链：

```
瘦≠好 → 研究 concluded → 肥胖对某些病 protective（反直觉！）
→ However 太瘦也有风险 → 锻炼 workouts → 大脑 functions 提升
→ 尾段：面对事实 → impact on health → compared with 正常体重
```

**你第一二段没懂 = 首句"瘦不一定好"这个反直觉定调没接住 → protective（保护性的）这种反直觉词全选错。**

### 错因分类

| 类型 | 涉及题 | 占比 |
|---|---|---|
| **段落逻辑没懂**（首段定调反直觉） | 1/2/3 | 3/8 |
| **词义/搭配不熟** | 5/7/8 | 3/8 |
| **固定搭配没记住** | 18/19 | 2/8 |

### Fix Plan · 完型三步法

| 步骤 | 操作 |
|---|---|
| ① **首句精读 + 翻译** | 把第一句写出中文，确定全文基调（正向/负向/反直觉） |
| ② **每空先判逻辑方向** | 前后句并列/转折/递进？→ 选方向一致的词 |
| ③ **固定搭配速查** | 不会的搭配当场查、当场记 |

### 必背固定搭配

| 搭配 | 含义 |
|---|---|
| conclude from studies | 从研究中**得出结论** |
| be protective against | 对……有**保护作用** |
| have an impact on | 对……有**影响** |
| compared with/to | 与……**相比** |
| it turns out that... | **结果是**…… |
| well-being | 健康**福祉**（≠ wealth ≠ welfare） |
