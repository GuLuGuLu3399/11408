# 错题日记 07 · 高等数学

> 配套 1000 题强化第 8 章（一元函数积分学的概念与性质）。题 1 熟练度问题不记；题 2/3/4 收黎曼和与比大小方法盲点；题 10/12 收反常积分判敛（假规则纠正 + $\ln$ 因子 $\varepsilon$ 比较）。

## [数学] 黎曼和极限——系数里的常数项别丢（强化 8.2）

### Question
$\displaystyle\lim_{n\to\infty}\sum_{k=1}^n\frac{1}{n}\!\left(\frac{2k-1}{2n}+1\right)f\!\left(\frac{2k-1}{2n}\right)$

### My Answer
**我的选择**：把系数抄成 $\frac{k-n/2}{n^2}$，按 $\int_0^1(x-\tfrac12)f\,dx$ 选 B。
**理由**：识破了 $\frac{2k-1}{2n}$ 是中点采样，但抄录把 "+1" 项弄丢，凑成 $(x-\tfrac12)$。

### Correct Answer
**C**：$\int_0^1(x+1)f(x)\,dx$。
**正解**：提 $\frac1n$ 后系数就是 $x_k+1$，$\frac1n\sum(x_k+1)f(x_k)\to\int_0^1(x+1)f\,dx$。

### Error Pattern
抄完题面没回原文核对，凭"应该是 $(x-\tfrac12)f$"的错觉把抄错版当成自己算的。**方法对、答案因抄录错。**

### Core Concept
黎曼和识别（中点采样 $\frac{2k-1}{2n}$）；系数与采样点差 $O(\frac1n)$ 被吸收。

### Fix Plan
抄完题面**用特殊项代入验算**：本题 $k=n$ 时系数应为 $(\frac{2n-1}{2n}+1)\frac1n\to\frac2n$，而 $\frac{k-n/2}{n^2}|_{k=n}=\frac1{2n}$——量级都对不上，立刻能发现抄错。

### 识别信号
$\sum$ 极限里 $\frac{2k-1}{2n}$、$\frac{k}{n}$、$\frac{k}{n}+c$ 都是采样点平移；**系数里跟 $k$ 无关的常数项原样保留，不自作主张改号。**

### 变式自测
系数改为 $\frac1n(\frac{2k-1}{2n}-\tfrac12)$，答：$\int_0^1(x-\tfrac12)f\,dx$。

## [数学] 对数型黎曼和——ln(1+u)~u 就是正解（强化 8.3）

### Question
$\displaystyle\lim_{n\to\infty}\sum_{i=0}^{n-1}\ln\!\left[1+\tfrac1n f\!\left(\tfrac in\right)\right]$

### My Answer
**我的选择**：用放缩做出 C，但一直在犹豫能否用 $\ln(1+x)\sim x$。
**理由**：不敢确信等价无穷小在"求和号内 + 每项 $\to0$"情形合法。

### Correct Answer
**C**：$\int_0^1 f(x)\,dx$。**正解就是 $\ln(1+u)\sim u$**（$u=\frac1n f(\frac in)\to0$），不是放缩。

### Error Pattern
对"等价无穷小能否进求和号"没把握，宁可绕放缩。其实**只要替换的是整项且每项 $u\to0$ 一致成立**，就和、积分可交换（Taylor：$\ln(1+u)=u+o(u)$，余项求和 $o(1)$）。

### Core Concept
等价无穷小在黎曼和中的合法性；$\ln(1+u),\ e^u-1,\ (1+u)^\alpha-1\sim u$（$u\to0$）。

### Fix Plan
见到 $\sum\ln(1+\text{小})$、$\sum(e^{\text{小}}-1)$、$\sum[(1+\text{小})^\alpha-1]$ → **第一直觉等价无穷小替换**，再转黎曼和。别绕放缩。

### 变式自测
$\lim_{n\to\infty}\sum_{i=1}^n[e^{\frac1n f(i/n)}-1]$，答：$\int_0^1 f\,dx$。

## [数学] 定积分比大小——区间再现求精确关系，不是定性比较（强化 8.4）

### Question
比较 $I_1=\int_0^{2\pi}\frac{\sin x}{x}dx$、$I_2=\int_0^{2\pi}\frac{\sin x}{2\pi-x}dx$、$I_3=\int_0^{2\pi}\frac{\sin x}{x(2\pi-x)}dx$。

### My Answer
**我的选择**：起初选 C（方向对），但提取解析时把 $I_2$ 写成 $=I_1$、答案改写成 B，把自己带偏。
**理由**：用区间再现想"统一分母比较"，没做到底（漏 $\sin(2\pi-t)=-\sin t$ 变号），停在前感觉。

### Correct Answer
**C**：$I_2<I_3<I_1$。
- $I_1>0$：在 $\pi$ 切开，第二段令 $t=x-\pi$，$I_1=\int_0^\pi\frac{\pi\sin x}{x(\pi+x)}dx>0$。
- $I_2=-I_1<0$：令 $t=2\pi-x$，$\sin(2\pi-t)=-\sin t$（**变号不能漏**）。
- $I_3=0$：部分分式 $\frac{1}{x(2\pi-x)}=\frac{1}{2\pi}(\frac1x+\frac{1}{2\pi-x})$，$I_3=\frac{I_1+I_2}{2\pi}=0$。

### Error Pattern
**区间再现只用来"统一分母定性比较"，没用来求精确代数关系。** 正确用法：替换后把两个积分变成"同一个积分 ± 自己"，直接定值（$I_2=-I_1$、$I_3=\frac{I_1+I_2}{2\pi}=0$），不用比较。

### Core Concept
区间再现 $x\to(a+b)-x$；部分分式把 $I_3$ 挂到 $I_1,I_2$；作差保号证符号。

### Fix Plan
**定积分比大小三步**：①区间对称含 $\sin$ → 先试 $x\to(a+b)-x$ 找 $I=\pm I$；②分母是积 → 部分分式挂到已知积分；③判符号 → 作差配分母证保号。**绝不凭感觉。**

### 识别信号
积分区间关于中点对称、被积函数含 $\sin$ 或分母是 $x(a+b-x)$ 形 → 上区间再现 + 部分分式。

### 变式自测
$I=\int_0^\pi\frac{x\sin x}{1+\cos^2 x}dx$，用 $x\to\pi-x$。答：$I=\frac{\pi^2}{4}$。

## [数学] 反常积分判敛——"两函数各收敛则积收敛"是假规则（强化 8.10）

### Question
$\displaystyle\int_0^1\frac{1}{x^p|\ln x|^q}\,dx$ 收敛的条件。

### My Answer
**我的选择**：把被积函数拆成 $\frac1{x^p}\cdot\frac1{|\ln x|^q}$，判断"每个因子各自的积分收敛 → 乘积收敛"。
**理由**：套了一条不存在的规则，且**漏看了 $x=1$ 也是奇点**（$\ln x\to0$）。

### Correct Answer
**A**：$p<1$ **且** $q<1$。
**正解**：双奇点（$x=0$ 与 $x=1$）换元统一。令 $\ln x=-t$，$x=e^{-t}$：
$$\int_0^1\frac1{x^p|\ln x|^q}dx=\int_0^{+\infty}\frac{e^{(p-1)t}}{t^q}dt=\underbrace{\int_0^1\frac{e^{(p-1)t}}{t^q}dt}_{\sim\,\int_0^1\frac1{t^q}\,\Rightarrow\,q<1}+\underbrace{\int_1^{+\infty}\frac{e^{(p-1)t}}{t^q}dt}_{\text{结论大观 ④：}p<1\text{ 必收敛}}$$
两段同时收敛 → $p<1$ 且 $q<1$。

### Error Pattern
**用了不存在的"两函数各收敛则积收敛"规则**（反例：$\int_0^1\frac1{x^{1/2}}\cdot\frac1{x^{1/2}}dx=\int_0^1\frac1x$ 发散，但每个因子积分都收敛）+ **漏双奇点**（只看 $x=0$，没看 $x=1$ 处 $\ln x\to0$）。

### Core Concept
双奇点积分必先换元统一；换元后化归 [[积分学强化笔记]] 结论大观的 ④ 型（$\int e^{-\alpha x}\cdot$ 多项式）。

### Fix Plan
**先把假规则从脑子里删掉**。判反常积分：①找全奇点（含 $\ln x$、$\frac1x$、$\frac1{1-x}$ 等）；②奇点 $\geqslant2$ 或被积函数含 $\ln x$ → 令 $\ln x=\pm t$ 换元；③套结论大观定参量范围。

### 变式自测
$\int_0^1\frac1{x^p(1-x)^q}dx$ 的收敛条件。答：$p<1$ 且 $q<1$（双奇点 $x=0$ 由 $p$、$x=1$ 由 $q$，分别套 ①a）。

## [数学] 反常积分判敛——ln 因子的 "+ε 比较法"（强化 8.12）

### Question
$\displaystyle\int_0^1\frac{\ln x}{x^p(1-x)^{1-p}}\,dx$ 收敛的条件。

### My Answer
**我的选择**：拆两边、各取极限、求交集——**方法对**，但 $x=0$ 端不会处理 $\ln x$，晕。
**理由**：思路正确（拆奇点 + 比较 + 交集），缺"$\ln x$ 怎么比"的那一招。

### Correct Answer
**A**：$-1<p<1$。
- $x\to0$：$\frac{\ln x}{x^p(1-x)^{1-p}}\sim\frac{\ln x}{x^p}$。因 $\ln x$ 比任何 $x^{-\epsilon}$ 慢，$\lim_{x\to0}\frac{\ln x/x^p}{1/x^{p+\epsilon}}=x^\epsilon\ln x\to0$，由 $\int_0\frac1{x^{p+\epsilon}}$ 收敛（$p+\epsilon<1$）→ 原积分收敛，得 $p<1$。
- $x\to1$：$\ln x\sim -(1-x)$，$\sim -(1-x)^p$，$\int\frac1{(1-x)^{-p}}$ 收敛需 $-p<1\Rightarrow p>-1$。
- 交集：$-1<p<1$。

### Error Pattern
方法对，**$x=0$ 端不会用 "+$\epsilon$ 比较"**。$\ln x$ 不是幂函数，没有干净阶数，要靠 $\epsilon$ 把它夹进幂函数框架。

### Core Concept
**$\ln x$ 比任何幂函数都慢**（$\lim x^\epsilon\ln x=0$）；处理 $\ln$ 因子的固定套路是 $+\epsilon$（或 $-\epsilon$）比较法。关联 [[积分学强化笔记]] 结论大观 ①b（$\int_0^1\frac{\ln x}{x^p}$ 收敛 $0\leqslant p<1$）。

### Fix Plan
判反常积分见到 $\ln x$ 因子：拿 $\frac1{x^{p\pm\epsilon}}$ 作比较，用极限比较法（$\lim=0$ 则被比较收敛 → 原收敛）。$\epsilon$ 取满足 $p\pm\epsilon<1$ 的小正数即可。

### 变式自测
$\int_0^1\frac{\ln x}{x^p}dx$ 收敛条件。答：$p<1$（$0\leqslant p<1$ 是反常情形套 ①b；$p<0$ 时 $x=0$ 非奇点，自然收敛）。
