# 错题日记 09 · 英语

> 2004 年考研英语一 Text 2「Alphabetism」——名字字母排序歧视。Q1（例证题）+ Q4（词义句意·ZZZ 文化符号），两题全错。

## 全文核心（理解框架 ✅）

幼儿园点名按 A-Z、AAAA 汽车比 Zodiac 占便宜、毕业典礼先叫 ABC 后叫 Zysmans → 核心论点：**alphabetism = 字母表排序造成的隐蔽不平等**（insidious inequality）。你的段落大意理解是对的——"对他们没好处"。

## Q1 · 例证题 🔴

### Question
What does the author intend to illustrate with AAAA cars and Zodiac cars?（AAAA 汽车和 Zodiac 汽车说明什么？）

### My Answer
**选了 B**。不认识 **conspicuous**，把 **overlook** 记成了"高看"。

### Correct Answer
**A**：A kind of **overlooked**（被忽视的）inequality。

→ 原文关键词 **insidious**（隐蔽的）≈ **overlooked** = 反义于 **conspicuous**（明显的）。

### 错因诊断

| 词 | 我当时以为 | 实际 |
|---|---|---|
| conspicuous | 不认识 | **明显的**、显眼的 |
| overlook | 高看 | **忽视**、漏掉 |
| insidious | — | **暗中为害的**、不易察觉的 |

→ **三个词构成考研英语的高频同反义替换组**：原文 insidious ⇔ overlooked（同义）、⇔ conspicuous（反义）。不认识 conspicuous → 无法用反义排除 B。

### 例证题套路

| 例证题铁律 | 说明 |
|---|---|
| 正确选项 = 论点 | 往前找**段首/前一段的中心句**，原文此处的论点是 "insidious inequality" |
| 错误选项常见坑 | **就事论事**——把例子本身当作论点（看到 AAAA/Zodiac 就选 brand discrimination） |
| 正确选项里不该有 | 例子里的具体词（car / brand / Zodiac）——那是例子，不是论点 |

### 识别信号
题干含 `illustrate / demonstrate / show / cite / quote / example` → 答案**不在例子本身，在往前那句论点**。例证题的正确选项一定是你**没读过例子时也能选出来的概括句**。

### 变式自测
同篇文章，如果问"the author mentions kindergarten roll-call to..."，答案仍在第一段论点（让名字靠后的孩子吃亏 = overlooked disadvantage），不在 kindergarten 本身。

## Q4 · 词义句意题 🔴

### Question
What does "most people are literally having a ZZZ" (Line 2, Paragraph 5) mean? 选 B。

定位句： _At university graduation ceremonies, the ABCs proudly get their awards first; by the time they reach the Zysmans **most people are literally having a ZZZ**._

### My Answer
**选了 D**（以为是拼字/猜谜游戏）。在不知道 ZZZ = 打鼾的前提下，凭"对他们没好处"的大意猜测的。

### Correct Answer
**B**：They are noisily dozing off（在打鼾 / 睡着了）。

| 选项 | 含义 | 为什么错 |
|---|---|---|
| A | 不耐烦 | 你替原文加了情感判断（"等太久 → 不耐烦"），原文没给出 impatient |
| **B** | 大声打鼾 | ✅ ZZZ = 漫画打鼾拟声词 |
| C | 羞辱感 | 张冠李戴——可能 Zysmans 有羞辱感，不是观众 |
| D | 猜字谜 | 望文生义——看到 ZZZ 联想拼写，没查上下文 |

### 错因诊断
**不认识 ZZZ 文化符号 + 没用上下文推导**（考研式第一次考查漫画拟声词，考场正确率仅 20%）。

你有两条（当时都没用）：
- **前半句的对比**："ABC 姓先领 → Zysmans 才轮到" → 时间拖很久 → 观众**已经在做别的事了**（睡觉）。如果只是"猜字谜"，时间轴对不上。
- **后文的同义线索**："lose interest as they plough through them"（越翻越没兴趣）→ 从"失去兴趣"能推出"走神 / 睡着"。

### 词义题套路

| 步骤 | 操作 |
|---|---|
| ①定位 | 找到该词/句的准确位置 |
| ②看**前一句** | 找出"什么导致了这件事" |
| ③看**后一句** | 找出"接下来发生了什么"，通常有同义/反向线索 |
| ④套选项 | 用②③得到的语境约束，逐个排除——不满足语境的一律刷掉 |

### 下次做词义题
别跳过上下文。**不认识一个词没关系——前后两句的词你都认识。** 用熟人推陌生人的位置。

## 本节词汇（必背）

| 词 | 释义 | 记忆钩子 |
|---|---|---|
| insidious | 暗中为害的、不易察觉的 | in + sid（=sit，坐在里面）→ 藏在内部搞事 |
| conspicuous | 明显的、显眼的 | con + spic（=看）→ 大家都看得见的 → **明显** |
| overlook | 忽视 | over + look → 从上面扫过 → 没仔细看 → 忽视 |
| prejudice | 偏见 | pre + jud（=judge）→ 提前判断 → 偏见 |
| bias | 偏见、偏向 | — |
| subtle | 微妙的、不易察觉的 | sub + tle → 细细的 → 不容易发现的 |

> **铁链**：reading passage 出现 insidious → 选项中必有 overlooked / subtle（同义）或 conspicuous / apparent（反义）。这组词一旦配齐，例证题的排除法飞快。
