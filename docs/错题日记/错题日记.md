# 错题日记

考研错题与复盘记录。每篇按科目归集多道错题，统一结构为：**题目 → 我的答案 → 正解 → 错因 → 核心概念 → 修正方案 → 变式自测**。

## 科目（subject）

- 计算机组成原理
- 数据结构
- 操作系统
- 计算机网络
- 高等数学

## 复习状态（status）

- 待复习
- 已掌握
- 需重学

## 组织约定

- 每篇日记可含多道错题，每道用 `## [科目] 标题` 起头，子段（Question / 正解 / 错因 …）用 `###`。
- 文件级 H1 命名为「错题日记 0N · 科目」，是 Tolaria 笔记列表显示的标题。
- 用 `related_to` 关联到相关 **概念笔记**（408笔记 / 考研数学），形成「概念 ↔ 错题」的双向复习网。
- 文末 `## 相关链接` 用 wikilink 列出相关概念笔记与同系列错题日记。
