# 错题日记 06 · 高等数学




## [数学] 1^∞ 型极限的通用化简法

### Question
已知极限条件：
$$ \lim_{x\to0} \left( 1 + x + \frac{f(x)}{x} \right)^{\frac{1}{x}} = e^3 $$

求以下极限的值：
$$ \lim_{x\to0} \left( 1 + \frac{f(x)}{x} \right)^{\frac{1}{x}} $$

### My Answer
**我的选择**：当时采用危险的取巧思维
**理由**：强行将 $1 + x + \frac{f(x)}{x}$ 类比为 $1 + 3x$，靠直觉猜想式子的形式。

### Correct Answer
**正确答案**：$e^2$
**正解**：对含有抽象函数的 $1^{\infty}$ 幂指函数，标准操作是利用恒等式 $u^v = e^{v \ln u}$ 将极限转移到指数位置。
1. 提取指数上的极限条件：$\text{原式} = e^{\lim_{x\to0} \frac{1}{x} \ln\left(1 + x + \frac{f(x)}{x}\right)} = e^3$，剥离出指数等式：$\lim_{x\to0} \frac{1}{x} \ln\left(1 + x + \frac{f(x)}{x}\right) = 3$。
2. 利用等价无穷小替换化简对数：由 $\ln(1+\square) \sim \square$，当 $x\to0$ 时 $\ln\left(1 + x + \frac{f(x)}{x}\right) \sim x + \frac{f(x)}{x}$，代入得 $\lim_{x\to0} \frac{x + \frac{f(x)}{x}}{x} = 3 \Rightarrow \lim_{x\to0}\left(1 + \frac{f(x)}{x^2}\right) = 3 \Rightarrow \lim_{x\to0} \frac{f(x)}{x^2} = 2$。由于分母趋于 0 时极限存在，分子必趋于 0，故暗含条件 $\lim_{x\to0} \frac{f(x)}{x} = 0$。
3. 代入目标极限求值：对所求极限同样恒等变形与等价替换，$\lim_{x\to0}\left(1 + \frac{f(x)}{x}\right)^{\frac{1}{x}} = e^{\lim_{x\to0} \frac{1}{x} \cdot \frac{f(x)}{x}} = e^{\lim_{x\to0} \frac{f(x)}{x^2}} = e^2$。

### Error Pattern
在处理 $1^{\infty}$ 型未定式极限时，过度依赖直觉"猜想"式子的形式（强行将 $1+x+\frac{f(x)}{x}$ 类比为 $1+3x$），是一种极其危险的"取巧"思维。遇到变量同时存在于底数和指数时，必须使用取对数或恒等变形化为指数形式，剥离抽象函数！

### Core Concept
- 未定式极限
- 等价无穷小替换
- 洛必达法则

### Expected Context
- **笔记路径**：函数、极限、连续、一元函数微分学
- **检索关键词**：$1^{\infty}$ 型未定式、凑 e 法则、恒等变形 $u^v = e^{v\ln u}$、等价无穷小 $\ln(1+\square) \sim \square$、抽象函数极限

### Fix Plan
凑 $e$ 法则：看到底数趋近于 1、指数趋近于 $\infty$，无脑套用公式 $\lim u^v = e^{\lim (u-1)v}$。不要凭借直觉去猜测抽象函数的多项式形式，极易在变体中翻车。

> **变式自测**：若已知 $\lim_{x\to0}(1+2x+f(x))^{\frac{1}{x}} = e^5$，求 $\lim_{x\to0}\frac{f(x)}{x^2}$。（提示：先凑 $e$ 提取指数，再用 $\ln(1+\square) \sim \square$ 化简。答案：3。）

## [数学] 指数函数与根式的等价无穷小替换

### Question
求下列极限：
$$ \lim_{x\to0} \frac{e - e^{\cos x}}{\sqrt[3]{1+x^2} - 1} $$

### My Answer
**我的选择**：当时对 $e^A - e^B$ 结构不敏感
**理由**：没有形成"提取公共项构造 $e^{\square} - 1$"的肌肉记忆；分母的广义二项式展开 $\sqrt[3]{1+x} - 1$ 也没有第一时间映射到对应的等价无穷小公式。

### Correct Answer
**正确答案**：$\frac{3}{2}e$
**正解**：本题是纯粹的等价无穷小代换基本功测试，核心在于分别对分子和分母进行降维构造。
1. 分子提取公因式构造标准型：遇到 $e^A - e^B$，先提取常数项 $e$，强行凑出 $1 - e^{\square}$ 或 $e^{\square} - 1$：$e - e^{\cos x} = e(1 - e^{\cos x - 1})$。由 $e^u - 1 \sim u$ 得 $1 - e^u \sim -u$；$\cos x - 1 \sim -\frac{1}{2}x^2$，故分子等价于 $e \cdot \left(\frac{1}{2}x^2\right)$。
2. 分母广义二项式等价替换：利用 $(1+x)^{\alpha} - 1 \sim \alpha x$，$\sqrt[3]{1+x^2} - 1 = (1+x^2)^{\frac{1}{3}} - 1 \sim \frac{1}{3}x^2$。
3. 比值求解：$\text{原式} = \lim_{x\to0} \frac{e \cdot \frac{1}{2}x^2}{\frac{1}{3}x^2} = \frac{3}{2}e$。

### Error Pattern
对于分子 $e^A - e^B$ 的结构极度不敏感，没有形成"提取公共项构造 $e^{\square} - 1$"的肌肉记忆。同时，分母的广义二项式展开 $\sqrt[3]{1+x} - 1$ 也没有第一时间映射到对应的等价无穷小公式。

### Core Concept
- 等价无穷小替换
- 泰勒公式

### Expected Context
- **笔记路径**：函数、极限、连续
- **检索关键词**：等价无穷小替换、$e^u - 1 \sim u$、$(1+x)^{\alpha} - 1 \sim \alpha x$、提取公因式、指差提底法则、$\cos x - 1 \sim -\frac{1}{2}x^2$

### Fix Plan
指差提底法则：见到 $a^{f(x)} - a^{g(x)}$，一律提取低次项（或极限趋于常数的项），构造出 $a^{\text{常数}}(a^{\Delta} - 1)$，然后使用 $a^u - 1 \sim u \ln a$ 一波带走。

> **同类题预警**：遇到类似 $\lim_{x\to0} \frac{2^x - 2^{\sin x}}{x^3}$ 时，第一步必须提取公因式 $2^{\sin x}$（或提取 $2^x$），构造出 $2^{x-\sin x} - 1$ 再结合泰勒公式替换。

> **变式自测**：求 $\lim_{x\to0}\frac{3^x - 3^{\tan x}}{x^3}$。（提示：提取 $3^{\tan x}$，构造 $3^{x-\tan x}-1$，再用 $a^u-1 \sim u\ln a$ 和 $x-\tan x$ 的泰勒展开。答案：$\frac{1}{3}\ln 3$。）

## [数学] 对数极限与泰勒展开确定阶数

### Question
已知：
$$ \lim_{x\to0} \frac{\ln\left(\frac{2+x}{2+\sin x}\right)}{x^n} = A \quad (\text{其中} A \text{为非零常数}) $$

求常数 $n$ 与 $A$ 的值。

### My Answer
**我的选择**：当时没有思路
**理由**：面对对数极限 $\ln\left(\frac{A}{B}\right)$ 时，没有意识到要通过"加1减1"的技巧构造出 $\ln(1+\square)$ 以便使用等价无穷小；对常见函数 $x - \sin x$ 的泰勒展开阶数记忆模糊，无法匹配分母的 $x^n$。

### Correct Answer
**正确答案**：$n = 3$，$A = \frac{1}{12}$
**正解**：解题核心路径：对数等价替换 → 提取常数项极限 → 泰勒展开匹配阶数。
1. 构造 $\ln(1+\square)$ 并等价替换：$\ln\left(\frac{2+x}{2+\sin x}\right) = \ln\left(1 + \frac{x-\sin x}{2+\sin x}\right)$。当 $x\to0$ 时 $\frac{x-\sin x}{2+\sin x} \to 0$，可直接等价替换为 $\frac{x-\sin x}{2+\sin x}$。
2. 分离非零极限因式：$\lim_{x\to0}(2+\sin x) = 2$ 是非零常数，可直接分离：分子极限 $\sim \frac{x-\sin x}{2}$。
3. 泰勒展开定阶：$\sin x \sim x - \frac{1}{6}x^3$，故 $x - \sin x \sim \frac{1}{6}x^3$。代回原式：$\lim_{x\to0} \frac{\frac{1}{12}x^3}{x^n} = A$。因 $A$ 为非零常数，分子分母必须同阶，故 $n = 3$，$A = \frac{1}{12}$。

### Error Pattern
面对对数极限 $\ln\left(\frac{A}{B}\right)$ 时没有意识到通过"加1减1"构造 $\ln(1+\square)$ 以使用等价无穷小；对 $x - \sin x$ 的泰勒展开阶数记忆模糊，无法匹配分母 $x^n$。

### Core Concept
- 等价无穷小替换
- 泰勒公式
- 阶的比较

### Expected Context
- **笔记路径**：函数、极限、连续、一元函数微分学
- **检索关键词**：对数极限、泰勒公式 $\sin x \sim x - \frac{1}{6}x^3$、$x - \sin x \sim \frac{1}{6}x^3$、同阶无穷小定阶、$\ln\left(\frac{A}{B}\right)$ 加1减1化简

### Fix Plan
对数分式化简口诀：对数里有分式，必定是"分子减分母，除以原分母，外面加个1"：$\ln\left(\frac{A}{B}\right) = \ln\left(1 + \frac{A-B}{B}\right) \sim \frac{A-B}{B}$。

> **同类题预警**：注意符号陷阱——如果题目是 $x - \tan x$，则泰勒展开的等价替换为 $-\frac{1}{3}x^3$，一正一负极易在计算 $A$ 时出错！

> **变式自测**：若改为 $\lim_{x\to0}\frac{\ln\left(\frac{1+x}{1+\sin x}\right)}{x^n} = A$（$A$ 非零），求 $n$ 和 $A$。（提示：分母极限为 1 而非 2。答案：$n = 3$，$A = \frac{1}{6}$。）

## [数学] 脱帽法（Peano 余项）反解抽象函数

### Question
设：
$$ \lim_{x\to0} \frac{\sin 2x + x f(x)}{x^3} = 1 $$

求极限：
$$ \lim_{x\to0} \frac{2\cos x + f(x)}{x^2} $$

### My Answer
**我的选择**：当时不会处理
**理由**：在已知极限求未知极限时，不会处理"孤立"的抽象函数 $f(x)$，完全忘记了利用带有 Peano 余项的等式关系来反解出 $f(x)$ 的多项式形式（即"脱帽法"）。

### Correct Answer
**正确答案**：$\frac{4}{3}$
**正解**："脱帽法"核心本质：极限等式 $\lim \frac{A}{B} = C \Leftrightarrow A = C \cdot B + o(B)$，由此可将极限转换为多项式恒等式。
1. 利用脱帽法反解含抽象函数的项：由原等式，分子可表示为分母的三阶等价：$\sin 2x + x f(x) = x^3 + o(x^3)$，移项得 $x f(x) = x^3 - \sin 2x + o(x^3)$。
2. 构造目标极限匹配多项式：目标极限分母是 $x^2$，分子是 $2\cos x + f(x)$，由于只知道 $x f(x)$ 的表达式，必须上下同乘 $x$：$\frac{2x\cos x + x f(x)}{x^3}$，将步骤一表达式代入分子 $= 2x\cos x + x^3 - \sin 2x + o(x^3)$。
3. 泰勒展开计算余项极限：$2x\cos x \sim 2x\left(1 - \frac{1}{2}x^2\right) = 2x - x^3$；$\sin 2x \sim 2x - \frac{(2x)^3}{6} = 2x - \frac{4}{3}x^3$；相减得 $(2x - x^3) - \left(2x - \frac{4}{3}x^3\right) = \frac{1}{3}x^3$，故 $L_2 = \frac{1}{3}$，最终结果 $= 1 + \frac{1}{3} = \frac{4}{3}$。

### Error Pattern
在已知极限求未知极限时，不会处理"孤立"的抽象函数 $f(x)$，完全忘记了利用带有 Peano 余项的等式关系来反解出 $f(x)$ 的多项式形式（脱帽法）。

### Core Concept
- Peano 余项
- 脱帽法
- 泰勒公式

### Expected Context
- **笔记路径**：函数、极限、连续、一元函数微分学
- **检索关键词**：Peano 余项、脱帽法 $A = CB + o(B)$、抽象函数反解、升阶强凑法、整体代换、泰勒展开 $\sin 2x$、$\cos x$

### Fix Plan
升阶强凑法：当已知条件能反解出 $x^k f(x)$ 时，所求极限中若出现 $f(x)$，不要去试图除以 $x$ 单独求 $f(x)$（会引入低阶 $o$ 的除法导致精度丢失），而是直接给所求极限上下同乘 $x^k$，强行凑出 $x^k f(x)$ 进行整体代换！

> **同类题预警**：若已知 $\lim_{x\to0}\frac{e^x - 1 - f(x)}{x^2} = 2$，则可以直接写出恒等式 $f(x) = e^x - 1 - 2x^2 + o(x^2)$。

> **变式自测**：已知 $\lim_{x\to0}\frac{e^x - 1 - x - f(x)}{x^3} = \frac{1}{6}$，求 $\lim_{x\to0}\frac{f(x)}{x^2}$。（提示：脱帽得 $f(x) = e^x - 1 - x - \frac{1}{6}x^3 + o(x^3)$，再用泰勒展开 $e^x$。答案：$\frac{1}{2}$。）
