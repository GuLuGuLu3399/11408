# 错题日记 12 · 英语

> 2002 考研英语一 Text 3「油价与经济衰退」。选 ACDAC，正确 BBACB，**0 对**。Day 3 晚间疲劳状态做题。

## 全文骨架（一段一句）

| 段 | 一句话 | 信号词 |
|---|---|---|
| P1 | **提问**：经济衰退会回来吗？OPEC 减产 → 油价涨 3 倍 → 想起 1973/1979 石油危机 | "scary memories" |
| P2 | **补充**：伊拉克暂停出口 + 北半球冬天 + 经济增长 → 短期推高 | "another push" |
| P3 | **核心转折**：这次不同——富裕经济体对石油**依赖降了 50%** | **"less dependent"** ← 全文关键词 |
| P4 | **数据**：OECD 估油价涨 50% → GDP 仅降 0.25%~0.5% | 数字 = 论据 |
| P5 | **更多理由**：无全面通胀 + 股市没崩 + 消费者信心在 | "not lose sleep" |

> **一句话文章**：油价涨了但不会引发衰退，因为对石油的依赖比 70 年代低太多了。**作者态度：乐观。**

## 长难句精拆（明天重点看）

> _"For every dollar of GDP (in constant prices) rich economies now use nearly 50% less oil than in 1973."_

```
主干：rich economies use 50% less oil（富裕经济体少用了 50% 的石油）
修饰①：for every dollar of GDP（每美元 GDP）
修饰②：in constant prices（不变价格——扣除通胀后的）
修饰③：than in 1973（与 1973 相比）
```

翻译：按不变价格算，富裕经济体现在每产出 1 美元 GDP 消耗的石油比 1973 年少了近 50%。

> **拆法**：先找主语+谓语+宾语 → 再挂修饰语。这句话的主干只有 6 个词。

## 背景知识（中国考生盲区）

| 概念 | 解释 |
|---|---|
| OPEC | 石油输出国组织，控制全球原油供应 |
| 1973 石油危机 | 第四次中东战争 → 阿拉伯国家减产 → 油价翻 4 倍 → 全球滞胀 |
| 1979-80 危机 | 伊朗革命 → 油价翻 3 倍 → 再次衰退 |
| 滞胀 | 经济停滞 + 通胀同时发生（70 年代特产） |
| OECD | 经济合作与发展组织 = 富裕国家俱乐部 |

> **这五个背景概念是理解全文的前提。** 不懂"石油危机"是什么 → 读 P1 就懵 → 后面全靠猜。
