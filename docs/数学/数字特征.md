# 数字特征

## 核心定义

数学期望 衡量分布的 **中心位置**，方差 衡量 **离散程度**，协方差 衡量 **共同波动**，相关系数 衡量标准化后的 **线性相关强弱**。

期望的基本公式：$E(X)=\sum_k x_kp_k$（离散）或 $E(X)=\int_{-\infty}^{+\infty}xf(x)\,dx$（连续）。

方差计算核心公式：$\operatorname{Var}(X)=E(X^2)-$ **$[E(X)]^2$**，永远非负。

期望线性性：$E(aX+b)=$ **$aE(X)+b$**；方差放缩：$\operatorname{Var}(aX+b)=$ **$a^2\operatorname{Var}(X)$**——常数 $b$ 不影响方差，系数要 **平方**。

协方差公式：$\operatorname{Cov}(X,Y)=E(XY)-$ **$E(X)E(Y)$**。

相关系数：$\rho_{XY}=$ **$\dfrac{\operatorname{Cov}(X,Y)}{\sqrt{\operatorname{Var}(X)\cdot\operatorname{Var}(Y)}}$**，取值范围 **$[-1,1]$**。

独立一定推出协方差为 **0**（不相关），但协方差为 0 **不能推出独立**。

## 关键细节 / 操作步骤

1. 第一步：先求 数学期望 $E(X)$。
2. 第二步：再由 $E(X^2)$ 和方差公式求 方差 $\operatorname{Var}(X)=$ **$E(X^2)-[E(X)]^2$**。
3. 第三步：多变量情形补充 协方差 与 相关系数。
4. 第四步：若题目给离散分布，直接按 **求和** 计算期望。
5. 第五步：若题目给连续分布，按 **积分** 计算期望。
6. 第六步：若题目问线性变换 $Y=aX+b$，优先套 **期望线性** 与 **方差平方放缩**。
7. 第七步：若题目问独立变量和的方差，用 $\operatorname{Var}(X+Y)=$ **$\operatorname{Var}(X)+\operatorname{Var}(Y)$**（独立时协方差为零）。
8. 第八步：若题目问协方差，先写 $E(XY)$ 再减去 **$E(X)E(Y)$**。
9. 第九步：若题目问相关系数，注意分母是 **标准差乘积** 不是方差乘积。
10. 第十步：若题目问多个随机变量线性组合的方差，先展开 $\operatorname{Var}(\sum a_iX_i)=\sum a_i^2\operatorname{Var}(X_i)+2\sum_{i<j}a_ia_j\operatorname{Cov}(X_i,X_j)$。

> **⚠️ 易错辨析**
>
> 1. $E(X^2)\neq [E(X)]^2$——前者是 **二阶矩**，后者是期望的平方，除非 $X$ 几乎为常数否则不等。
> 2. $\operatorname{Var}(aX+b)$ 不是乘 $a$，而是乘 **$a^2$**，因为方差是二次运算。
> 3. 协方差为 0 只意味着没有 **线性相关**，不能推出独立——反例：$X\sim N(0,1)$，$Y=X^2$，则 $\operatorname{Cov}(X,Y)=0$ 但 $X,Y$ 不独立。
> 4. 期望是 **线性算子**，方差不是线性的，不能乱拆：$\operatorname{Var}(X+Y)=\operatorname{Var}(X)+\operatorname{Var}(Y)+2\operatorname{Cov}(X,Y)$。
> 5. 相关系数 $\rho=\pm 1$ 表示 **完全线性相关**，不是"完全相关"。

> **💡 技巧与口诀**
> 口诀：**先线性求期望，再平方求方差；协方差看共同波动，相关系数看标准化后强弱**。
>
> 应用场景：一看到"线性变换、独立和、波动程度、相关强弱"，就把数字特征作为第一工具。
>
> 若题目给 $aX+b$，直接套 $E(Y)=aE(X)+b$ 和 $\operatorname{Var}(Y)=a^2\operatorname{Var}(X)$。
>
> 相关系数需要标准化是因为协方差会受 **量纲** 影响。

> **📝 真题闭环**
> 设 $X$ 与 $Y$ 独立，$E(X)=2$，$E(Y)=3$，$E(X^2)=5$，$E(Y^2)=10$，求 $\operatorname{Var}(2X-Y)$。
>
> **解题思路**：审题抓"独立""方差""线性组合"，切入点是 **方差展开公式**；先算 $\operatorname{Var}(X)=E(X^2)-[E(X)]^2=$ **$5-4=1$**，$\operatorname{Var}(Y)=E(Y^2)-[E(Y)]^2=$ **$10-9=1$**；独立时协方差为 0，则 $\operatorname{Var}(2X-Y)=$ **$4\operatorname{Var}(X)+\operatorname{Var}(Y)$** $=4+1=5$。
>
> 答案：**$\operatorname{Var}(2X-Y)=4\times1+1=5$**。
