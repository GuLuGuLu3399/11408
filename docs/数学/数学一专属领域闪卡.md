# 数学一专属领域闪卡

## 梯度与方向导数

### 核心定义

方向导数 是标量函数在某点沿指定方向的变化率，而 梯度 是所有方向导数中取到**最大值**的那个方向。设 $u = f(x,y,z)$ 在点 $P_0$ 处可微，沿方向 $\mathbf{l} = (\cos\alpha, \cos\beta, \cos\gamma)$ 的方向导数为：

$$\frac{\partial f}{\partial \mathbf{l}} = \frac{\partial f}{\partial x}\cos\alpha + \frac{\partial f}{\partial y}\cos\beta + \frac{\partial f}{\partial z}\cos\gamma$$

梯度本身是一个**向量**，其表达式为：

$$\text{grad } u = \nabla f = \left(\frac{\partial f}{\partial x},\ \frac{\partial f}{\partial y},\ \frac{\partial f}{\partial z}\right)$$

二者的核心关系：$\frac{\partial f}{\partial \mathbf{l}} =$ **$\nabla f \cdot \mathbf{l}$**，即方向导数等于梯度在目标方向上的投影。

### 关键细节

1. **梯度方向**：函数值增长最快的方向；梯度的模 $|\nabla f|$ 就是该点最大的方向导数
2. **梯度与等值面正交**：在 $f(x,y,z) = c$ 的等值面上，梯度方向就是该点的法向量方向
3. **方向导数为零**：当 $\mathbf{l} \perp \nabla f$ 时，沿该方向函数值不变（沿等值面走）
4. **Nabla 算子** $\nabla = \left(\frac{\partial}{\partial x},\ \frac{\partial}{\partial y},\ \frac{\partial}{\partial z}\right)$ 是场论所有运算的基础——梯度是对标量的**作用（算子直接作用于标量函数）**，散度是对向量的**点乘**，旋度是对向量的**叉乘**

> **⚠️ 方向导数存在的条件**
> 方向导数 $\neq$ 偏导数。偏导数是沿坐标轴方向的方向导数。此外，函数在某点沿某方向的方向导数存在，**不能**推出函数在该点可微。但反过来，函数可微 $\Rightarrow$ 沿任意方向的方向导数都存在，且可用梯度公式计算。

> **💡 物理直觉速记**
> 想象站在山坡上（标量场=海拔），梯度指向最陡的上坡方向（向量的方向），梯度的模长就是最陡的坡度（最大的方向导数）。沿等高线走，方向导数为零——不上不下。

> **📝 真题闭环**
> 设 $u = \ln(x^2 + y^2 + z^2)$，求 $\text{grad } u$ 在点 $P(1,2,-2)$ 处的值及其模长。
>
> **解题思路**：先求三个偏导 → 代入点坐标 → 模长 = 偏导数平方和开根号。
> $\frac{\partial u}{\partial x} = \frac{2x}{x^2+y^2+z^2}$，同理求 $y, z$ 偏导。代入 $P(1,2,-2)$，$x^2+y^2+z^2 = 9$。
>
> $\text{grad } u\big|_P =$ **$\left(\frac{2}{9},\ \frac{4}{9},\ -\frac{4}{9}\right)$**，$|\text{grad } u|_P =$ **$\frac{2\sqrt{2}}{9}$**

## 散度与高斯公式的场论基础

### 核心定义

散度 是 向量场 在某一点的"发散程度"，它将向量场压缩为一个**标量**。设向量场 $\mathbf{F} = (P, Q, R)$，则散度定义为：

$$\text{div } \mathbf{F} = \nabla \cdot \mathbf{F} = \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}$$

物理意义：散度描述空间中某点是"源"还是"汇"。$\text{div } \mathbf{F} > 0$ 表示该点是**源（向外喷发）**，$\text{div } \mathbf{F} < 0$ 表示该点是**汇（向内吸收）**，$\text{div } \mathbf{F} = 0$ 表示该点既不喷也不吸——称为 无源场。

### 关键细节

1. **散度定理（高斯公式的微分形式）**：$\oiint_{\Sigma} \mathbf{F} \cdot d\mathbf{S} = \iiint_{\Omega} \text{div } \mathbf{F}\, dV$，即通量 = 散度的体积分
2. **$\text{div}(\text{grad } u) = \Delta u = \nabla^2 u$**：拉普拉斯算子，即梯度的散度。$\Delta u = \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} + \frac{\partial^2 u}{\partial z^2}$
3. **散度的线性**：$\text{div}(\alpha \mathbf{F} + \beta \mathbf{G}) = \alpha\,\text{div } \mathbf{F} + \beta\,\text{div } \mathbf{G}$
4. **$\text{div}(u\mathbf{F}) = u\,\text{div } \mathbf{F} + \text{grad } u \cdot \mathbf{F}$**（乘积法则，考研常考）

> **⚠️ 散度 vs 通量**
> 散度是**逐点**的标量值；通量是**整个曲面**上的积分结果。高斯公式把二者联系起来。题目若问"某点是否为源"，算散度；若问"通过曲面的总流量"，算通量（再用高斯公式转体积分）。

> **💡 散度速算口诀**
> 散度就是对每个分量求各自偏导再相加：$P_x' + Q_y' + R_z'$。注意是对各自的变量求偏导（$P$ 对 $x$，$Q$ 对 $y$，$R$ 对 $z$），千万别交叉！

> **📝 真题闭环**
> 设 $\mathbf{F} = (x^2y,\ yz^2,\ -xz)$，求 $\text{div } \mathbf{F}$ 及其在点 $(1,1,1)$ 处的值。
>
> **解题思路**：分别求 $\frac{\partial P}{\partial x}$、$\frac{\partial Q}{\partial y}$、$\frac{\partial R}{\partial z}$ → 相加 → 代入点坐标。
>
> $\frac{\partial}{\partial x}(x^2y) = 2xy$，$\frac{\partial}{\partial y}(yz^2) = z^2$，$\frac{\partial}{\partial z}(-xz) = -x$
>
> $\text{div } \mathbf{F} = 2xy + z^2 - x$，代入 $(1,1,1)$ 得 $\text{div } \mathbf{F}\big|_{(1,1,1)} =$ **$2$**

## 旋度与保守场判定

### 核心定义

旋度 描述 向量场 在某一点的旋转程度，结果是一个**向量**。设 $\mathbf{F} = (P, Q, R)$，旋度是 Nabla 算子与 $\mathbf{F}$ 的叉乘：

$$\text{rot } \mathbf{F} = \nabla \times \mathbf{F} = \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \\ \frac{\partial}{\partial x} & \frac{\partial}{\partial y} & \frac{\partial}{\partial z} \\ P & Q & R \end{vmatrix}$$

展开后三个分量为：$\left(\frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z},\ \frac{\partial P}{\partial z} - \frac{\partial R}{\partial x},\ \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right)$

当 $\nabla \times \mathbf{F} = \mathbf{0}$ 时，$\mathbf{F}$ 称为 无旋场，等价条件包括：$\mathbf{F}$ 是 保守场、曲线积分与路径无关、$\mathbf{F} =$ **$\text{grad } u$**（某势函数的梯度）。

### 关键细节

1. **旋度为零的判定**：只需验证三个分量对应偏导之差是否为零
2. **旋度的散度恒为零**：$\text{div}(\text{rot } \mathbf{F}) \equiv 0$，即旋度场一定是无源场
3. **斯托克斯公式的核心**：环流量 $\oint_{\Gamma} \mathbf{F} \cdot d\mathbf{r} = \iint_{\Sigma} (\nabla \times \mathbf{F}) \cdot d\mathbf{S}$，即旋度的面积分 = 沿边界的环流量
4. **保守场的实用判定**（单连通区域内）：$\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$，$\frac{\partial P}{\partial z} = \frac{\partial R}{\partial x}$，$\frac{\partial Q}{\partial z} = \frac{\partial R}{\partial y}$，三组偏导相等即可

> **⚠️ 旋度展开的记忆陷阱**
> 行列式展开时，$\mathbf{i}$ 分量是 $\frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z}$，**不是** $\frac{\partial Q}{\partial y} - \frac{\partial R}{\partial z}$。注意是"交叉"取偏导——$\mathbf{i}$ 分量涉及 $R$ 对 $y$ 和 $Q$ 对 $z$，与 $x$ 无关。用口诀："**不在同一行同一列**"。

> **💡 旋度展开口诀**
> $\mathbf{i}$ 分量：$R_y' - Q_z'$；$\mathbf{j}$ 分量：$P_z' - R_x'$；$\mathbf{k}$ 分量：$Q_x' - P_y'$。规律：每个分量都是"**另外两个函数对各自变量的交叉偏导之差**"，且 $\mathbf{j}$ 分量符号取反。

> **📝 真题闭环**
> 判定向量场 $\mathbf{F} = (yz,\ xz,\ xy)$ 是否为保守场。若是，求势函数 $u$ 使得 $\mathbf{F} = \text{grad } u$。
>
> **解题思路**：算旋度三个分量 → 全为零则保守 → 积分求势函数。
> $\frac{\partial(xy)}{\partial y} - \frac{\partial(xz)}{\partial z} = x - x = 0$（$\mathbf{i}$ 分量）
> 同理 $\mathbf{j}, \mathbf{k}$ 分量均为零 → 保守场。
> 令 $u_x' = yz$，积分得 $u = xyz + g(y,z)$；再对 $y$ 求偏导与 $Q = xz$ 对比 → 势函数 $u =$ **$xyz + C$**

## 三重积分与坐标系转换

### 核心定义

三重积分 $\iiint_{\Omega} f(x,y,z)\,dV$ 是二重积分向三维空间的推广。当 $f$ 表示密度时，三重积分的物理意义就是空间区域 $\Omega$ 的总质量。计算的核心在于选择合适的坐标系，将 $dV$ 转化为对应的体积元素。

### 关键细节

1. **直角坐标（穿线法/投影法）**：
   - 先确定 $\Omega$ 在 $xOy$ 面上的投影区域 $D_{xy}$
   - 穿线确定 $z$ 的范围：$z_1(x,y) \leq z \leq z_2(x,y)$
   - $\iiint_{\Omega} f\,dV = \iint_{D_{xy}} \left[\int_{z_1}^{z_2} f\,dz\right] dxdy$

2. **柱面坐标**（出现 $x^2 + y^2$ 或圆柱面时首选）：
   - 代换：$x = r\cos\theta$，$y = r\sin\theta$，$z = z$
   - 体积元素：$dV =$ **$r\,dr\,d\theta\,dz$**（**永远不要漏乘 $r$**）

3. **球面坐标**（出现 $x^2 + y^2 + z^2$ 或球体时首选）：
   - 代换：$x = r\sin\varphi\cos\theta$，$y = r\sin\varphi\sin\theta$，$z = r\cos\varphi$
   - 体积元素：$dV =$ **$r^2\sin\varphi\,dr\,d\varphi\,d\theta$**
   - $\varphi$ 是与 $z$ 轴正方向的夹角，范围 $[0, \pi]$；$\theta$ 是 $xOy$ 面上的极角

4. **对称性速判**：若 $\Omega$ 关于 $xOy$ 面对称且 $f$ 关于 $z$ 是奇函数，则积分为零

> **⚠️ 坐标系选择的红线**
> - 看到 $x^2 + y^2$ → **柱面坐标**
> - 看到 $x^2 + y^2 + z^2$ → **球面坐标**
> - 其他 → 直角坐标
> 千万别对球体用直角坐标硬算，积分限复杂且容易出错。球面坐标的 $r^2\sin\varphi$ 是由雅可比行列式得出的，必须死记。

> **💡 球面坐标积分限口诀**
> 球心在原点的球体：$r$ 从 $0$ 到 $R$；$\varphi$ 从 $0$ 到 $\pi$（经线扫半圈）；$\theta$ 从 $0$ 到 $2\pi$（纬线转一圈）。对于上半球，$\varphi$ 从 $0$ 到 $\pi/2$；对于锥体，$\varphi$ 从 $0$ 到半锥角。

> **📝 真题闭环**
> 计算 $\iiint_{\Omega} (x^2 + y^2 + z^2)\,dV$，其中 $\Omega$ 为球体 $x^2 + y^2 + z^2 \leq R^2$。
>
> **解题思路**：球体 → 球面坐标。被积函数 $x^2+y^2+z^2 = r^2$，体积元素 $r^2\sin\varphi\,dr\,d\varphi\,d\theta$。
> 积分 $= \int_0^{2\pi} d\theta \int_0^{\pi} \sin\varphi\,d\varphi \int_0^{R} r^2 \cdot r^2\,dr$
> $= 2\pi \cdot 2 \cdot \frac{R^5}{5}$
>
> 结果 $=$ **$\frac{4\pi R^5}{5}$**

## 曲线积分：第一类与第二类

### 核心定义

曲线积分 分为两类，核心区别在于**有没有方向**。

**第一类**（对弧长）：$\int_L f(x,y,z)\,ds$，物理意义为变密度曲线的质量。$ds$ 是弧长微元，与路径方向**无关**。

**第二类**（对坐标）：$\int_L P\,dx + Q\,dy + R\,dz$，物理意义为变力沿曲线做的功。与路径方向**有关**，反向取负号。

### 关键细节

1. **第一类曲线积分的计算**（参数方程 $x=x(t), y=y(t), z=z(t)$，$t: \alpha \to \beta$）：
   $$ds = \sqrt{x'(t)^2 + y'(t)^2 + z'(t)^2}\,dt$$
   $$\int_L f\,ds = \int_{\alpha}^{\beta} f(x(t),y(t),z(t))\sqrt{x'^2+y'^2+z'^2}\,dt$$
   注意：$\alpha < \beta$（下限必须小于上限，因为弧长恒正）

2. **第二类曲线积分的计算**：
   $$\int_L P\,dx + Q\,dy + R\,dz = \int_{\alpha}^{\beta}\left[Px'(t) + Qy'(t) + Rz'(t)\right]dt$$
   注意：$\alpha$ 对应起点，$\beta$ 对应终点（方向敏感）

3. **第一类转第二类的桥梁**（方向余弦法）：
   $$\int_L P\,dx = \int_L P\cos\alpha\,ds,\quad \int_L Q\,dy = \int_L Q\cos\beta\,ds$$

4. **格林公式**（平面闭曲线 $L$ 正向包围区域 $D$）：
   $$\oint_L P\,dx + Q\,dy = \iint_D \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right)dxdy$$

> **⚠️ 两类积分的核心差异**
> 第一类：下限 $<$ 上限（弧长恒正），与方向无关。第二类：下限对应起点，上限对应终点，反向积分变号。考试时先判断是哪一类，再确定积分限——这是丢分重灾区。

> **💡 格林公式速用条件**
> (1) $L$ 必须闭合（不闭合要补线再减）；(2) $L$ 取正向（逆时针）；(3) $P, Q$ 在 $D$ 内有连续偏导。若 $D$ 内有奇点（如分母为零），需挖去奇点再用格林公式。

> **📝 真题闭环**
> 计算 $\oint_L (x^2 - y)\,dx + (y^2 + x)\,dy$，其中 $L$ 为圆周 $x^2 + y^2 = 1$ 取正向。
>
> **解题思路**：闭曲线 → 格林公式。$P = x^2 - y$，$Q = y^2 + x$，$\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} = 1 - (-1) = 2$。
> 原式 $= \iint_D 2\,dxdy = 2 \times \text{面积} = 2 \times \pi \times 1^2$
>
> 结果 $=$ **$2\pi$**

## 曲面积分：第一类与第二类

### 核心定义

曲面积分 是曲线积分从"一根线"到"一张面"的升维。同样分为两类。

**第一类**（对面积）：$\iint_{\Sigma} f(x,y,z)\,dS$，物理意义为变密度曲面的质量。$dS$ 是面积微元，与曲面**侧**无关。

**第二类**（对坐标）：$\iint_{\Sigma} P\,dydz + Q\,dzdx + R\,dxdy$，物理意义为向量场穿过曲面的**通量（流量）**。与曲面侧有关，取反侧积分变号。

### 关键细节

1. **第一类曲面积分计算**（曲面 $z = z(x,y)$，投影到 $D_{xy}$）：
   $$dS = \sqrt{1 + (z_x')^2 + (z_y')^2}\,dxdy$$
   $$\iint_{\Sigma} f\,dS = \iint_{D_{xy}} f(x,y,z(x,y))\sqrt{1+z_x'^2+z_y'^2}\,dxdy$$

2. **第二类曲面积分计算**（以 $dxdy$ 项为例）：
   $$\iint_{\Sigma} R\,dxdy = \pm\iint_{D_{xy}} R(x,y,z(x,y))\,dxdy$$
   正负号取决于：法向量朝上（$z$ 分量 $>0$）取正，朝下取负

3. **第一类转第二类的桥梁**：
   $$\iint_{\Sigma} R\,dxdy = \iint_{\Sigma} R\cos\gamma\,dS$$
   其中 $\cos\gamma$ 是法向量与 $z$ 轴夹角的余弦

4. **高斯公式**（闭曲面 $\Sigma$ 取外侧，包围区域 $\Omega$）：
   $$\oiint_{\Sigma} P\,dydz + Q\,dzdx + R\,dxdy = \iiint_{\Omega}\left(\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}\right)dV$$

> **⚠️ 第二类曲面积分的侧**
> "上侧"指法向量与 $z$ 轴正方向成锐角（$\cos\gamma > 0$），"外侧"指法向量指向封闭体的外部。题目说"上侧"只针对 $dxdy$ 项——$dydz$ 项看 $x$ 方向，$dzdx$ 项看 $y$ 方向。如果曲面不封闭，需要**补面**使其封闭，算完高斯公式再减去补面的积分。

> **💡 dS 公式速记**
> $dS = \sqrt{1 + z_x'^2 + z_y'^2}\,dxdy$，同理若投影到 $yOz$ 面：$dS = \sqrt{1 + x_y'^2 + x_z'^2}\,dydz$。选择投影面时，投影区域越简单越好。

> **📝 真题闭环**
> 计算 $\iint_{\Sigma} (x^2 + y^2)\,dS$，其中 $\Sigma$ 为锥面 $z = \sqrt{x^2+y^2}$，$0 \leq z \leq 1$。
>
> **解题思路**：第一类曲面积分，$z = \sqrt{x^2+y^2}$，$z_x' = \frac{x}{\sqrt{x^2+y^2}}$，$z_y' = \frac{y}{\sqrt{x^2+y^2}}$。
> $z_x'^2 + z_y'^2 = 1$，所以 $dS = \sqrt{1+1}\,dxdy = \sqrt{2}\,dxdy$。
> 投影区域 $D_{xy}$：$x^2+y^2 \leq 1$。用极坐标：
> $\sqrt{2}\int_0^{2\pi}d\theta\int_0^1 r^2 \cdot r\,dr = \sqrt{2} \cdot 2\pi \cdot \frac{1}{4}$
>
> 结果 $=$ **$\frac{\sqrt{2}\pi}{2}$**

## 三大定理：格林、高斯、斯托克斯

### 核心定义

三大定理建立了**不同维度积分之间的桥梁**，核心思想是"边界上的积分 = 内部区域上导数的积分"。

**格林公式（2D 线→面）**：平面闭曲线积分 $\Leftrightarrow$ 二重积分

$$\oint_L P\,dx + Q\,dy = \iint_D\left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right)dxdy$$

**高斯公式（3D 面→体）**：闭曲面积分 $\Leftrightarrow$ 三重积分（考频最高）

$$\oiint_{\Sigma} P\,dydz + Q\,dzdx + R\,dxdy = \iiint_{\Omega}\left(\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}\right)dV$$

**斯托克斯公式（3D 线→面）**：空间闭曲线积分 $\Leftrightarrow$ 曲面积分

$$\oint_{\Gamma} P\,dx + Q\,dy + R\,dz = \iint_{\Sigma}\begin{vmatrix} dydz & dzdx & dxdy \\ \frac{\partial}{\partial x} & \frac{\partial}{\partial y} & \frac{\partial}{\partial z} \\ P & Q & R \end{vmatrix}$$

### 关键细节

1. **适用条件对比**：

| 定理 | 维度 | 边界要求 | 方向约定 |
|:-----|:-----|:---------|:---------|
| 格林 | 2D | $L$ 闭合 + 正向（逆时针） | 人沿 $L$ 走，区域在左手边 |
| 高斯 | 3D | $\Sigma$ 闭合 + 外侧 | 法向量指向体外 |
| 斯托克斯 | 3D | $\Gamma$ 闭合 + 曲面 $\Sigma$ 以其为边 | **右手螺旋定则** |

2. **补面/补线技巧**：
   - 格林：$L$ 不闭合时补直线段 $l$，$\oint = \int_L + \int_l$，算完减去 $\int_l$
   - 高斯：$\Sigma$ 不闭合时补平面 $\Sigma_1$ 封口，$\oint = \iint_{\Sigma} + \iint_{\Sigma_1}$，算完减去 $\iint_{\Sigma_1}$
   - 斯托克斯：$\Gamma$ 不闭合时补曲线段

3. **高斯公式的散度核心**：$\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}$ 就是散度 $\text{div } \mathbf{F}$

4. **斯托克斯公式的旋度核心**：行列式展开后就是 $(\nabla \times \mathbf{F}) \cdot \mathbf{n}$ 在 $\Sigma$ 上的积分

> **⚠️ 三大定理的闭合红线**
> 每个定理都要求边界**闭合**。若题目给的曲线/曲面不闭合，必须先补成闭合的。补面法是数一必考的套路：**先补面 → 用公式 → 减去补面的积分**。补面的选择原则是让补面上的积分尽量好算（通常选平行于坐标面的平面）。

> **💡 定理选择决策树**
> 遇到积分题先看：积分对象是**线**还是**面**？区域是否**闭合**？
> - 闭曲线 + 平面 → 格林
> - 闭曲线 + 空间 → 斯托克斯
> - 闭曲面 + 空间 → 高斯
> - 不闭合 → 补面/补线后再选定理

> **📝 真题闭环**
> 计算 $\oiint_{\Sigma} x\,dydz + y\,dzdx + z\,dxdy$，其中 $\Sigma$ 为球面 $x^2+y^2+z^2 = R^2$ 取外侧。
>
> **解题思路**：闭曲面 + 外侧 → 高斯公式。$P=x, Q=y, R=z$。
> 散度 $= \frac{\partial x}{\partial x} + \frac{\partial y}{\partial y} + \frac{\partial z}{\partial z} = 3$
> 原式 $= \iiint_{\Omega} 3\,dV = 3 \times \frac{4}{3}\pi R^3$
>
> 结果 $=$ **$4\pi R^3$**
