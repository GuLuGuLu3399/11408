# 多元函数微分学

> 18 讲第 13 讲（RAG 提取 + 用户 Day 5 听课）。内容量大，核心是**二重极限 vs 累次极限**和**概念关系图**。

## 一、二重极限与累次极限（RAG 第 13 讲·第四部分）

### 定义

| 类型 | 定义 | 记号 |
|---|---|---|
| **二重极限** | $(x,y)\to(x_0,y_0)$ 以**任意路径**趋近时 $f\to A$ | $\lim_{\substack{x\to x_0\\y\to y_0}}f(x,y)=A$ |
| **累次极限** | 先固定一个变量求极限，再对另一个求极限 | $\lim_{x\to x_0}[\lim_{y\to y_0}f(x,y)]$ |

### 三条关系定理（必背）

| 条件 | 结论 |
|---|---|
| 二重存在 + 累次存在 | **二者必相等** |
| 两个累次都存在但**不等** | **二重极限不存在**（判据！） |
| 二重存在 | 累次**未必**存在 |
| 累次存在 | 二重**未必**存在 |

> **考试杀手锏**：**两个累次极限不等 → 二重极限不存在**。这是判二重极限不存在的最常用方法。

### 例 13.6（RAG）

$f(x,y)=\frac{x^2-y^2}{x^2+y^2}$，$I_1=\lim_{x\to0}[\lim_{y\to0}f]=1$，$I_2=\lim_{y\to0}[\lim_{x\to0}f]=-1$。
$I_1\neq I_2$ → **二重极限不存在**。选 (A)。

## 二、概念关系图——连续/可偏导/可微/可导（RAG 第 13 讲·第二部分核心）

> 这是多元函数微分学最核心的选择题考点——**六个概念之间的推出与反例**。

### 推出关系

```
偏导数连续 → 可微 → 连续
               ↗     ↘
         可偏导（两个偏导都存在）→ 沿坐标轴方向连续
```

### 关键定理

| 推出 | 结论 | 反例方向 |
|---|---|---|
| **偏导数连续** → **可微** | 成立 | 偏导数不连续也可能可微（需验证 $\lim\frac{\Delta z-dz}{\rho}=0$） |
| **可微** → **连续** | 成立 | 连续不一定可微 |
| **可微** → **可偏导** | 成立 | 可偏导不一定可微 |
| **可微** → **沿任意方向可导** | 成立 | — |
| 连续 → 可偏导 | **不成立** | $f=\sqrt{|xy|}$ 连续但偏导不存在 |
| 可偏导 → 连续 | **不成立** | 一元可导→连续，多元可偏导**不一定连续**！ |
| 可偏导 → 可微 | **不成立** | 偏导存在但不可微：$f=\frac{xy}{\sqrt{x^2+y^2}}$ 在原点 |

### 可微的验证步骤（概念关系图的关键操作）

$$\Delta z = f(x_0+\Delta x, y_0+\Delta y) - f(x_0,y_0)$$
$$dz = f_x'(x_0,y_0)\Delta x + f_y'(x_0,y_0)\Delta y$$
$$\lim_{\rho\to 0}\frac{\Delta z - dz}{\rho} = 0\ (\rho=\sqrt{(\Delta x)^2+(\Delta y)^2})$$

> 若极限 = 0 → **可微**；若 ≠ 0 或不存在 → **不可微**。

## 三、全微分的计算（$D_1$ 常规操作）

$$dz = f_x'(x,y)\,dx + f_y'(x,y)\,dy$$

二阶全微分：$d^2z = f_{xx}''\,dx^2 + 2f_{xy}''\,dxdy + f_{yy}''\,dy^2$

> 当 $f_{xy}''=f_{yx}''$（混合偏导连续时），二阶交叉项合并为 $2f_{xy}''$。

## 四、多元函数性质研究——三向解题法框架（RAG 第 13 讲·第二部分）

| 武器 | 操作 |
|---|---|
| 连续 | 验证 $\lim f(x,y) = f(x_0,y_0)$ |
| 偏导数 | 固定一个变量，对另一个求导 |
| 全微分 | 计算 $dz = f_x\,dx + f_y\,dy$ |
| 概念关系图 | 写 $\Delta z$ → 写 $dz$ → 验证 $\lim\frac{\Delta z-dz}{\rho}=0$ |
| 无条件极值 | 求驻点（$f_x=f_y=0$）→ $\Delta=AC-B^2$ 判别 |
| 条件极值 | 拉格朗日乘数法 |
| 隐函数求导 | $F(x,y,z)=0$ → $z_x'=-F_x'/F_z'$ |
| 复合函数求导 | 链式法则画依赖树 |

## 五、考研高频反例（选择题陷阱）

| 函数 | 性质 | 用途 |
|---|---|---|
| $f=\frac{xy}{x^2+y^2}$（原点设 0） | 可偏导但不连续 | 反例：可偏导 ↛ 连续 |
| $f=\sqrt{\|xy\|}$ | 连续但偏导不存在 | 反例：连续 ↛ 可偏导 |
| $f=\frac{xy}{\sqrt{x^2+y^2}}$（原点设 0） | 可偏导但不可微 | 反例：可偏导 ↛ 可微 |
| $f=\frac{x^2-y^2}{x^2+y^2}$ | 两个累次极限不等 | 反例：二重极限不存在 |

> **识别信号**：选择题问"以下哪个结论正确"出现连续/可偏导/可微混搭 → **先画概念关系图，再用反例排除**。

## 六、第 13 讲后半·应用深化（Day 6 补充）

> 计算密集区，四块：**复合函数二阶偏导**、**二元泰勒**、**闭区域最值**、**变量代换化简偏微分方程**。

### 1. 复合函数二阶偏导——再链式是命门 🔴

设 $z=f(u,v)$，$u=u(x,y)$，$v=v(x,y)$。一阶：
$$z_x = f_1 \cdot u_x + f_2 \cdot v_x$$

**关键**：$f_1, f_2$ 仍是 $u,v$ 的函数，对 $x$ 再求导时**必须再链式一次**：
$$\frac{\partial}{\partial x}[f_1] = f_{11}\,u_x + f_{12}\,v_x, \qquad \frac{\partial}{\partial x}[f_2] = f_{21}\,u_x + f_{22}\,v_x$$

> **最大坑**：把 $\partial[f_1]/\partial x$ 只写成 $f_{11}u_x$，**漏掉 $f_{12}v_x$** → 全题错。
> **救命简化**：$f$ 二阶连续偏导 $\Rightarrow$ $f_{12}=f_{21}$，交叉项对称合并，省一半计算。

### 2. 二元泰勒公式

在 $(x_0,y_0)$ 处展开（记 $h=x-x_0,\ k=y-y_0$），截到二阶：
$$f(x_0+h,\,y_0+k) = f_0 + (f_x h + f_y k) + \tfrac{1}{2}\!\left(f_{xx}h^2 + 2f_{xy}hk + f_{yy}k^2\right) + R$$

> **用途**：近似计算（截二阶估误差）、证不等式（展成多项式再放缩）。数一考过用泰勒证 $f(x,y)\ge 0$ 这类题。

### 3. 闭区域最值——别漏边界

最值 $=\max/\min\big\{$**内部驻点值** $\cup$ **边界最值**$\big\}$。

| 步骤 | 操作 |
|---|---|
| ① 内部 | 解 $f_x=f_y=0$，Hessian 筛内部极值 |
| ② 边界 | 边界当**条件极值**（拉格朗日）或**参数化**成一元求最值 |
| ③ 比较 | 内部极值 + 边界最值，取最大 / 最小 |

> **丢分重灾区**：只算内部驻点就交卷，**漏掉边界**。如圆域 $x^2+y^2\le 1$，边界 $x^2+y^2=1$ 必须单独处理。

### 4. 变量代换化简偏微分方程（多元微分 × 微分方程）

给 $z=f(u,v)$，$u=\varphi(x,y)$，$v=\psi(x,y)$，把关于 $x,y$ 的偏微分方程用 $u,v$ 表示并化简。

> **套路**：用链式把 $z_x, z_{xx}$ 写成 $f_1,f_2$ 的组合 → 代入原 PDE → 整理成关于 $u,v$ 的方程。本质就是**二阶偏导再链式的应用**，所以常和第 1 块捆在一起考。

## 核心定义

偏导数 是固定其他变量、对某一变量求导：$f_x = \lim_{h \to 0}\frac{f(x_0+h,y_0) - f(x_0,y_0)}{h}$。全微分 给出函数的局部线性近似：$df =$ **$f_x\,dx + f_y\,dy$**。方向导数 表示沿单位向量 $\mathbf{u}$ 方向的变化率：$D_{\mathbf{u}}f =$ **$\nabla f \cdot \mathbf{u}$**。梯度 指向函数增长最快方向：$\nabla f =$ **$(f_x, f_y)$**。

多元微分的核心逻辑链：**偏导数连续 $\Rightarrow$ 可微 $\Rightarrow$ 偏导存在且连续 $\Rightarrow$ 方向导数存在**，但偏导存在并不自动推出可微。可微 的充要条件是 $\lim_{(h,k)\to(0,0)}\frac{f(x_0+h,y_0+k) - f(x_0,y_0) - f_x h - f_y k}{\sqrt{h^2+k^2}} =$ **$0$**。

极值判别：先找 驻点（$f_x = f_y = 0$），再用 Hessian矩阵 判别。记 $A =$ **$f_{xx}$**，$B =$ **$f_{xy}$**，$C =$ **$f_{yy}$**，判别量 $\Delta =$ **$AC - B^2$**。当 $\Delta > 0$ 且 $A > 0$ 为 **极小值**；$\Delta > 0$ 且 $A < 0$ 为 **极大值**；$\Delta < 0$ 为 **鞍点（非极值）**；$\Delta = 0$ 时 **无法判定**，需另寻方法。

拉格朗日乘子法 用于约束极值问题：在约束 $g(x,y)=0$ 下求 $f(x,y)$ 的极值，构造 $F(x,y,\lambda) =$ **$f(x,y) + \lambda g(x,y)$**，解方程组 $\begin{cases} F_x = 0 \\ F_y = 0 \\ F_\lambda = 0 \end{cases}$ 即 **$f_x + \lambda g_x = 0$**，**$f_y + \lambda g_y = 0$**，**$g(x,y) = 0$**。

隐函数定理：若 $F(x,y)$ 在 $(x_0,y_0)$ 附近连续可微且 $F(x_0,y_0) = 0$、$F_y(x_0,y_0) \neq 0$，则存在 $y = y(x)$ 满足 $F(x,y(x)) = 0$，且 $\frac{dy}{dx} =$ **$-\frac{F_x}{F_y}$**。切平面方程：$z - z_0 =$ **$f_x(x_0,y_0)(x-x_0) + f_y(x_0,y_0)(y-y_0)$**，法向量 $\mathbf{n} =$ **$(f_x, f_y, -1)$**。

## 关键细节 / 操作步骤

1. 第一步：计算 一阶偏导 $f_x, f_y$，注意链式法则在复合函数中的使用。
2. 第二步：判断可微性——验证 **全微分极限** 是否为零，分段函数重点检查 **拼接点**。
3. 第三步：方向导数计算：先确认 $\mathbf{u}$ 是 **单位向量**（否则必须单位化），再代入 $D_{\mathbf{u}}f = \nabla f \cdot \mathbf{u}$。
4. 第四步：最速上升方向为 **梯度方向**，最速下降方向为 **负梯度方向**，方向导数最大值为 $|\nabla f|$。
5. 第五步：无条件极值：解 $f_x = 0, f_y = 0$ 得驻点，再用 Hessian 矩阵的 $AC - B^2$ 分类判别。
6. 第六步：条件极值：构造 **拉格朗日函数 $F = f + \lambda g$**，解三元方程组得候选点。
7. 第七步：隐函数求导：直接用公式 $\frac{dy}{dx} =$ **$-\frac{F_x}{F_y}$**，前提是 $F_y \neq 0$。
8. 第八步：切平面与法线：由偏导数值代入切平面方程，法线方程为 $\frac{x-x_0}{f_x} = \frac{y-y_0}{f_y} =$ **$\frac{z-z_0}{-1}$**。
9. 第九步：混合偏导 $f_{xy}$ 与 $f_{yx}$ 在 **偏导连续** 的条件下相等。
10. 第十步：闭区域最值问题：先找内部驻点，再检查 **边界极值**（常用拉格朗日法处理边界），最后比较所有候选点。

> **⚠️ 易错辨析**
> **偏导存在 $\neq$ 可微。** 反例：$f(x,y) = \frac{xy}{x^2+y^2}$（$f(0,0)=0$）在原点偏导均为零，但沿 $y=kx$ 方向极限为 $\frac{k}{1+k^2}$，不一致，故 **不可微**。**方向导数公式中 $\mathbf{u}$ 必须是单位向量**——反例：方向 $(3,4)$ 必须单位化为 $(3/5, 4/5)$。**可微 $\Rightarrow$ 连续，但连续 $\not\Rightarrow$ 可微**。**Hessian 判别 $\Delta = 0$ 时失效**——反例：$f(x,y) = x^4 + y^4$ 在 $(0,0)$ 处 $\Delta = 0$ 但取极小值，需用其他方法。**拉格朗日乘子法解出的点只是候选点**，还需比较函数值确定最大/最小。

> **💡 技巧与口诀**
> 口诀：**先偏后全，先梯度后方向，极值先驻点再判型，约束上乘子**。应用场景：题目出现"方向导数、最速上升、切平面"按梯度框架处理；出现"约束条件下极值"立刻构造拉格朗日函数。**可微性检验**：先尝试全微分形式，再做极限 $\lim \frac{\Delta f - df}{\rho}$ 是否为零。**二元极值快速判断**：算 $AC - B^2$，正看 $A$ 的符号，负是鞍点，零要另寻方法。**隐函数求导**比全微分法更直接，但全微分法不容易漏项。

> **📝 真题闭环**
> 求函数 $f(x,y) = x^3 - y^3 + 3x^2 + 3y^2 - 9x$ 的极值。

**解题思路**：审题抓"求极值"，切入点是 **找驻点再 Hessian 判别**；方法选择为无条件极值标准流程；计算关键点：$f_x = 3x^2 + 6x - 9 = 3(x-1)(x+3) = 0 \Rightarrow x =$ **$1$** 或 $x =$ **$-3$**，$f_y = -3y^2 + 6y = -3y(y-2) = 0 \Rightarrow y =$ **$0$** 或 $y =$ **$2$**，得四个驻点；$A = f_{xx} =$ **$6x+6$**，$B = f_{xy} = 0$，$C = f_{yy} =$ **$-6y+6$**；逐点计算 $AC - B^2$ 判别；易错防范是遗漏驻点或 Hessian 符号判断错误。

答案：**驻点 $(1,0)$：$A=12>0, C=6, \Delta=72>0$，极小值 $f=-5$；$(-3,0)$：$A=-12, C=6, \Delta=-72<0$，鞍点；$(1,2)$：$A=12, C=-6, \Delta=-72<0$，鞍点；$(-3,2)$：$A=-12<0, C=-6, \Delta=72>0$，极大值 $f=31$**
