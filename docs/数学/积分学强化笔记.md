# 积分学强化：黎曼和与定积分比大小

> 配套 1000 题强化第 8 章 + 张宇高数 18 讲·强化篇第 8 讲「一元函数积分学的概念与性质」。
> 第一二三点来自 1000 题 8.1–8.4 实战；第四五点从 18 讲第 8 讲 RAG 提炼（结论大观 + 具体型判敛）。

## 一、n 项和极限 → 黎曼和 → 定积分定义

### 本质
$\displaystyle\sum_{k=1}^n \tfrac1n\,g(x_k)\to\int_0^1 g(x)\,dx$，$x_k$ 是 $[0,1]$ 的采样点。考研题把"识别"做难：$x_k$ 和系数都伪装过。

### 常见采样点（识别信号）
- $\frac{k}{n}$：右端点
- $\frac{2k-1}{2n}$：中点
- $\frac{k}{n+c}$：略偏移（$O(\frac1n)$ 差被吸收）
- $\frac{2i}{2n+1}$、$\frac{i}{n+1}$ 等：都是采样点

### 三大易错
1. **$0\cdot\infty$ 误判**：每项 $\to0$、项数 $\to\infty$，整体**不会**自动 $\to0$。这恰是黎曼和的本质——结果是积分（有限非零）。反例：$\lim\frac1n\cdot n=1$。
2. **系数与采样点差 $O(\frac1n)$**：如系数用 $\frac{k}{n}$、采样点是 $\frac{2k-1}{2n}$，差 $\frac1{2n}\to0$，**不影响极限**，直接当 $x_k$。
3. **抄错系数的常数/符号项**：用特殊项（$k=n$）代入验算量级，立刻发现。

### 求和号里的等价无穷小
$\sum\ln(1+\text{小})$、$\sum(e^{\text{小}}-1)$、$\sum[(1+\text{小})^\alpha-1]$ → **先等价无穷小替换**（$\sim$ 小），再转黎曼和。合法因 $\ln(1+u)=u+o(u)$，余项求和 $o(1)$。

### 分离变量法
求和号里某部分**只含 $n$ 不含 $k$** → 提出来单独等价无穷小，剩下 $\sum f(x_k)$ 才是黎曼和。
例：$\sum(1-\cos\frac{\pi}{\sqrt n})\cdot\frac{1}{1+\cos(\pi i/2n)}$，分子提出 $1-\cos\frac\pi{\sqrt n}\sim\frac{\pi^2}{2n}$，分母和 $\to n\cdot\frac2\pi$，积 $=\pi$。

## 二、定积分比大小

### 三步法
1. **区间再现** $x\to(a+b)-x$：把积分变成"自己 ± 自己"。出现 $I=\pm I$ → 立刻定值（$I=0$ 或 $2I=$ 某值）。判 $\sin$ 变号：$\sin((a+b)-x)$ 在区间关于中点对称时**变号**。
2. **部分分式分解**：分母是 $x(a+b-x)$ 形 → $\frac1{x(a+b-x)}=\frac1{a+b}(\frac1x+\frac1{a+b-x})$，把 $I_3$ 挂到 $I_1,I_2$。
3. **作差保号**：判 $\int f$ 符号 → 拆成 $\int(\frac1x-\frac1{a+x})\sin x\,dx=\int\frac{a\sin x}{x(a+x)}dx$，证被积函数保号。

### 经典范式
$I_1=\int_0^{2\pi}\frac{\sin x}{x}$、$I_2=\int_0^{2\pi}\frac{\sin x}{2\pi-x}$、$I_3=\int_0^{2\pi}\frac{\sin x}{x(2\pi-x)}$：
- $I_2=-I_1$（区间再现，$\sin$ 变号）
- $I_3=\frac{I_1+I_2}{2\pi}=0$（部分分式）
- $I_1>0$（$\pi$ 处切开 + 作差保号）
- 排序 $I_2<I_3<I_1$。

## 三、识别信号速查

| 题面特征 | 第一反应 |
|---|---|
| $\sum_{n\to\infty}$ + $\frac{k}{n}$ 系数 | 黎曼和 → 定积分 |
| $\sum\ln(1+\cdot)$ / $e^{\cdot}-1$ | 等价无穷小替换再黎曼和 |
| 求和号里只含 $n$ 的因子 | 提出来单独等价无穷小 |
| 积分区间对称 + 含 $\sin$ | 区间再现找 $I=\pm I$ |
| 分母是 $x(a+b-x)$ | 部分分式挂到已知积分 |
| 判积分符号 | 切开 + 作差 + 配分母保号 |

## 四、含参反常积分敛散性结论大观（18 讲第 8 讲·第一部分）

### 速度原理（判敛总钥匙）
盯奇点处"无穷的速度"：
- **瑕积分**（盯 $x\to0^+$）：$x^p$ 趋 0 的速度决定 $\frac1{x^p}$ 趋 $+\infty$ 的速度 → $0<p<1$ 收敛，$p\geqslant1$ 发散。
- **无穷区间**（盯 $x\to+\infty$）：$x^p$ 趋 $+\infty$ 的速度决定 $\frac1{x^p}$ 趋 0 的速度 → $p>1$ 收敛，$p\leqslant1$ 发散。

### 结论表（背熟，考场直接套）

**无穷区间型：**

| 积分 | 收敛 | 发散 |
|---|---|---|
| ②a $\int_1^\infty \frac1{x^p}$ | $p>1$ | $p\leqslant1$ |
| ②b $\int_1^\infty \frac{\ln x}{x^p}$ | $p>1$ | $p\leqslant1$ |
| ⑥ $\int_2^\infty \frac1{x^p\ln^q x}$ | $p>1$(q 任意) 或 $p=1,q>1$ | $p<1$(q 任意) 或 $p=1,q\leqslant1$ |
| ④ $\int e^{-\alpha x}\ln^p x$、$e^{-\alpha x}x^q$ | $\alpha>0$ 必收敛 | $\alpha<0$ 必发散 |
| ⑩ $\int_2^\infty \frac1{\|\ln x\|^p}$ | — | 必发散 |

**瑕积分型：**

| 积分 | 收敛 | 发散 |
|---|---|---|
| ①a $\int_0^1 \frac1{x^p}$ | $0<p<1$ | $p\geqslant1$ |
| ①b $\int_0^1 \frac{\ln x}{x^p}$ | $0\leqslant p<1$ | $p\geqslant1$ |
| ⑤ $\int_1^2 \frac1{x^p\ln^q x}$ | $0<q<1$（与 p 无关） | $q\geqslant1$ |
| ⑦ $\int_0^1 \frac1{\|\ln x\|^p}$ | $p<1(p\neq0)$ | $p\geqslant1$ |
| ⑧ $\int_1^2 \frac1{\|\ln x\|^p}$ | $0<p<1$ | $p\geqslant1$ |
| ⑨ $\int_0^2 \frac1{\|\ln x\|^p}$ | $p<1(p\neq0)$ | $p\geqslant1$ |

### 等价代换推广（速度一样 → 结论一样）
- $x\to0$：$\sin x\sim x$ → $\int_0^1\frac1{\sin^p x}$ 同 ①a；$\ln(1+x)\sim x$、$\arctan x\sim x$ 同理。
- $x\to+\infty$，$a>0$：$ax+b$ 与 $x$ 同速 → $\int_1^\infty\frac1{(ax+b)^p}$ 同 ②a。
- 增长速度链：$\ln^p x \ll x^q \ll a^x \ll x^x$（$p,q>0,a>1$）——这是 ④ 的根据。

### 化归经典形式六招（$D_{23}$）
遇到含参积分先化归，再套上表：
1. **积分可拆性**：$\int_a^b=\int_a^c+\int_c^b$（$a,b$ 均为奇点时必拆）。
2. **分母设置法**：$x^p=\frac1{x^{-p}}$，$\ln^q x=\frac1{\ln^{-q}x}$，作比化标准形。
3. **换元法**：$1-x=t$、$\ln x=t$（把 $|\ln|$ 型化成 $\frac1{t^p}$）。
4. **分部积分**：把震荡因子（$\sin,\cos$）降一次幂。
5. **等价代换**：$\ln(1+x)\sim x$，$\arctan x\sim x$。
6. **加绝对值 → 绝对值判敛法**：$\int\|f\|$ 收敛 $\Rightarrow\int f$ 收敛（由 $0\leqslant f+\|f\|\leqslant2\|f\|$ + 比较判敛法证）。

### 经典例：$\int_1^\infty\frac{\sin x}{x}dx$ 收敛
分部积分 $=\cos1-\int_1^\infty\frac{\cos x}{x^2}dx$，而 $\int_1^\infty\|\frac{\cos x}{x^2}\|\leqslant\int_1^\infty\frac1{x^2}=1$ 绝对收敛 → 原积分收敛。

## 五、判断具体型反常积分的敛散性（18 讲第 8 讲·第二部分）

**三步**：①等价代换/换元化归 → ②套上面基本结论表 → ③定敛散。

**例 8.7** $\int_0^1 x^p(1-x)^{p-1}\ln x\,dx$ 收敛 ⇒ 选 (B) $-1<p\leqslant0$。
- 拆两奇点 $x=0$（$x^p\ln x$，套 ①b 型）与 $x=1$（$(1-x)^{p-1}\cdot\ln x\sim-(1-x)^p$），分别定 $p$ 的范围再取交集。
