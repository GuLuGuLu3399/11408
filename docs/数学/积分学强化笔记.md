# 积分学强化：黎曼和与定积分比大小

> 配套 1000 题强化第 8 章 + 张宇高数 18 讲·强化篇第 8 讲「一元函数积分学的概念与性质」。
> 第一二三点来自 1000 题 8.1–8.4 实战；第四五点从 18 讲第 8 讲 RAG 提炼（结论大观 + 具体型判敛）。

## 一、n 项和极限 → 黎曼和 → 定积分定义

### 本质
$\displaystyle\sum_{k=1}^n \tfrac1n\,g(x_k)\to\int_0^1 g(x)\,dx$，$x_k$ 是 $[0,1]$ 的采样点。考研题把"识别"做难：$x_k$ 和系数都伪装过。

### 常见采样点（识别信号）
- $\frac{k}{n}$：右端点
- $\frac{2k-1}{2n}$：中点
- $\frac{k}{n+c}$：略偏移（$O(\frac1n)$ 差被吸收）
- $\frac{2i}{2n+1}$、$\frac{i}{n+1}$ 等：都是采样点

### 三大易错
1. **$0\cdot\infty$ 误判**：每项 $\to0$、项数 $\to\infty$，整体**不会**自动 $\to0$。这恰是黎曼和的本质——结果是积分（有限非零）。反例：$\lim\frac1n\cdot n=1$。
2. **系数与采样点差 $O(\frac1n)$**：如系数用 $\frac{k}{n}$、采样点是 $\frac{2k-1}{2n}$，差 $\frac1{2n}\to0$，**不影响极限**，直接当 $x_k$。
3. **抄错系数的常数/符号项**：用特殊项（$k=n$）代入验算量级，立刻发现。

### 求和号里的等价无穷小
$\sum\ln(1+\text{小})$、$\sum(e^{\text{小}}-1)$、$\sum[(1+\text{小})^\alpha-1]$ → **先等价无穷小替换**（$\sim$ 小），再转黎曼和。合法因 $\ln(1+u)=u+o(u)$，余项求和 $o(1)$。

### 分离变量法
求和号里某部分**只含 $n$ 不含 $k$** → 提出来单独等价无穷小，剩下 $\sum f(x_k)$ 才是黎曼和。
例：$\sum(1-\cos\frac{\pi}{\sqrt n})\cdot\frac{1}{1+\cos(\pi i/2n)}$，分子提出 $1-\cos\frac\pi{\sqrt n}\sim\frac{\pi^2}{2n}$，分母和 $\to n\cdot\frac2\pi$，积 $=\pi$。

## 二、定积分比大小

### 三步法
1. **区间再现** $x\to(a+b)-x$：把积分变成"自己 ± 自己"。出现 $I=\pm I$ → 立刻定值（$I=0$ 或 $2I=$ 某值）。判 $\sin$ 变号：$\sin((a+b)-x)$ 在区间关于中点对称时**变号**。
2. **部分分式分解**：分母是 $x(a+b-x)$ 形 → $\frac1{x(a+b-x)}=\frac1{a+b}(\frac1x+\frac1{a+b-x})$，把 $I_3$ 挂到 $I_1,I_2$。
3. **作差保号**：判 $\int f$ 符号 → 拆成 $\int(\frac1x-\frac1{a+x})\sin x\,dx=\int\frac{a\sin x}{x(a+x)}dx$，证被积函数保号。

### 经典范式
$I_1=\int_0^{2\pi}\frac{\sin x}{x}$、$I_2=\int_0^{2\pi}\frac{\sin x}{2\pi-x}$、$I_3=\int_0^{2\pi}\frac{\sin x}{x(2\pi-x)}$：
- $I_2=-I_1$（区间再现，$\sin$ 变号）
- $I_3=\frac{I_1+I_2}{2\pi}=0$（部分分式）
- $I_1>0$（$\pi$ 处切开 + 作差保号）
- 排序 $I_2<I_3<I_1$。

## 三、识别信号速查

| 题面特征 | 第一反应 |
|---|---|
| $\sum_{n\to\infty}$ + $\frac{k}{n}$ 系数 | 黎曼和 → 定积分 |
| $\sum\ln(1+\cdot)$ / $e^{\cdot}-1$ | 等价无穷小替换再黎曼和 |
| 求和号里只含 $n$ 的因子 | 提出来单独等价无穷小 |
| 积分区间对称 + 含 $\sin$ | 区间再现找 $I=\pm I$ |
| 分母是 $x(a+b-x)$ | 部分分式挂到已知积分 |
| 判积分符号 | 切开 + 作差 + 配分母保号 |

## 四、含参反常积分敛散性结论大观（18 讲第 8 讲·第一部分）

### 速度原理（判敛总钥匙）
盯奇点处"无穷的速度"：
- **瑕积分**（盯 $x\to0^+$）：$x^p$ 趋 0 的速度决定 $\frac1{x^p}$ 趋 $+\infty$ 的速度 → $0<p<1$ 收敛，$p\geqslant1$ 发散。
- **无穷区间**（盯 $x\to+\infty$）：$x^p$ 趋 $+\infty$ 的速度决定 $\frac1{x^p}$ 趋 0 的速度 → $p>1$ 收敛，$p\leqslant1$ 发散。

### 结论表（背熟，考场直接套）

**无穷区间型：**

| 积分 | 收敛 | 发散 |
|---|---|---|
| ②a $\int_1^\infty \frac1{x^p}$ | $p>1$ | $p\leqslant1$ |
| ②b $\int_1^\infty \frac{\ln x}{x^p}$ | $p>1$ | $p\leqslant1$ |
| ⑥ $\int_2^\infty \frac1{x^p\ln^q x}$ | $p>1$(q 任意) 或 $p=1,q>1$ | $p<1$(q 任意) 或 $p=1,q\leqslant1$ |
| ④ $\int e^{-\alpha x}\ln^p x$、$e^{-\alpha x}x^q$ | $\alpha>0$ 必收敛 | $\alpha<0$ 必发散 |
| ⑩ $\int_2^\infty \frac1{\|\ln x\|^p}$ | — | 必发散 |

**瑕积分型：**

| 积分 | 收敛 | 发散 |
|---|---|---|
| ①a $\int_0^1 \frac1{x^p}$ | $0<p<1$ | $p\geqslant1$ |
| ①b $\int_0^1 \frac{\ln x}{x^p}$ | $0\leqslant p<1$ | $p\geqslant1$ |
| ⑤ $\int_1^2 \frac1{x^p\ln^q x}$ | $0<q<1$（与 p 无关） | $q\geqslant1$ |
| ⑦ $\int_0^1 \frac1{\|\ln x\|^p}$ | $p<1(p\neq0)$ | $p\geqslant1$ |
| ⑧ $\int_1^2 \frac1{\|\ln x\|^p}$ | $0<p<1$ | $p\geqslant1$ |
| ⑨ $\int_0^2 \frac1{\|\ln x\|^p}$ | $p<1(p\neq0)$ | $p\geqslant1$ |

### 等价代换推广（速度一样 → 结论一样）
- $x\to0$：$\sin x\sim x$ → $\int_0^1\frac1{\sin^p x}$ 同 ①a；$\ln(1+x)\sim x$、$\arctan x\sim x$ 同理。
- $x\to+\infty$，$a>0$：$ax+b$ 与 $x$ 同速 → $\int_1^\infty\frac1{(ax+b)^p}$ 同 ②a。
- 增长速度链：$\ln^p x \ll x^q \ll a^x \ll x^x$（$p,q>0,a>1$）——这是 ④ 的根据。

### 化归经典形式六招（$D_{23}$）
遇到含参积分先化归，再套上表：
1. **积分可拆性**：$\int_a^b=\int_a^c+\int_c^b$（$a,b$ 均为奇点时必拆）。
2. **分母设置法**：$x^p=\frac1{x^{-p}}$，$\ln^q x=\frac1{\ln^{-q}x}$，作比化标准形。
3. **换元法**：$1-x=t$、$\ln x=t$（把 $|\ln|$ 型化成 $\frac1{t^p}$）。
4. **分部积分**：把震荡因子（$\sin,\cos$）降一次幂。
5. **等价代换**：$\ln(1+x)\sim x$，$\arctan x\sim x$。
6. **加绝对值 → 绝对值判敛法**：$\int\|f\|$ 收敛 $\Rightarrow\int f$ 收敛（由 $0\leqslant f+\|f\|\leqslant2\|f\|$ + 比较判敛法证）。

### 经典例：$\int_1^\infty\frac{\sin x}{x}dx$ 收敛
分部积分 $=\cos1-\int_1^\infty\frac{\cos x}{x^2}dx$，而 $\int_1^\infty\|\frac{\cos x}{x^2}\|\leqslant\int_1^\infty\frac1{x^2}=1$ 绝对收敛 → 原积分收敛。

## 五、判断具体型反常积分的敛散性（18 讲第 8 讲·第二部分）

**三步**：①等价代换/换元化归 → ②套上面基本结论表 → ③定敛散。

**例 8.7** $\int_0^1 x^p(1-x)^{p-1}\ln x\,dx$ 收敛 ⇒ 选 (B) $-1<p\leqslant0$。
- 拆两奇点 $x=0$（$x^p\ln x$，套 ①b 型）与 $x=1$（$(1-x)^{p-1}\cdot\ln x\sim-(1-x)^p$），分别定 $p$ 的范围再取交集。

> 反常积分判敛是 18 讲第 8 讲重点——本轮 4 题（黎曼和 + 比大小）未涉及，**务必单开一轮刷判敛题**。

## 五补、一元函数积分学的计算（18 讲第 9 讲）

> ⚠️ RAG 重新向量化后第 9 讲已完整命中（例 9.1–9.11 + 变限求导型分类）。下方结合 RAG 提取 + 考纲补充。

### 张宇三向解题法框架（原书结构，RAG 提取 + 用户补充）

```
求一元函数的积分
├─ O(盯住目标)——先问"求什么"
├─ 基本积分公式（D₁ 常规操作 + D₂₃ 化归经典形式）
├─ 第一类换元法 / 凑微分法（D₁ + D₂₃）
├─ 第二类换元法（D₁ + D₂₃）
├─ 分部积分法（D₁ + D₂₃）
├─ 有理函数的积分（D₁ + D₂₃）
├─ 三角有理式的积分法（D₁ + D₂₃）
├─ 求出原函数并计算积分值（D₁）
├─ 变限积分函数的求导（D₁ + D₂₂转换等价表述 + D₂₃）
├─ 分段函数的积分（D₁ + D₂₃）
└─ 几何法（D₁ + D₂₃）
```

> **张宇符号体系**：
> - **O** = 盯住目标（先判断"这道题要我求什么"）
> - **D₁** = 常规操作（标准技巧，如基本公式、换元、分部）
> - **D₂₂** = 转换等价表述（换一种写法，如 $\int f \leftrightarrow$ 变上限求导）
> - **D₂₃** = 化归经典形式（把陌生问题归约为已知模型）

### 两类基本积分法

| 方法 | 适用结构 | 核心操作 |
|---|---|---|
| **换元积分法** | 复合结构 $\int f(g(x))g'(x)\,dx$ | 令 $t=g(x)$，$dt=g'(x)dx$ |
| **分部积分法** | 乘积结构 $\int u\,dv$ | $\int u\,dv=uv-\int v\,du$ |

**分部积分 $u$ 的选取口诀**（反对幂指三）：
$$\underbrace{\text{反三角}}_{\text{优先作 }u} > \underbrace{\text{对数}} > \underbrace{\text{幂函数}} > \underbrace{\text{三角}} > \underbrace{\text{指数}}_{\text{优先作 }dv}$$

### 有理函数积分

$$\frac{P(x)}{Q(x)} \xrightarrow{\text{部分分式分解}} \frac{A}{x-a}+\frac{B}{(x-a)^2}+\frac{Cx+D}{x^2+px+q}\ (p^2<4q)$$

| 分母因式 | 对应部分分式 | 积分结果 |
|---|---|---|
| $(x-a)$ | $\frac{A}{x-a}$ | $A\ln\|x-a\|$ |
| $(x-a)^n$ | $\frac{A}{(x-a)^n}$ | $\frac{-A}{(n-1)(x-a)^{n-1}}$ |
| $x^2+px+q$（不可约） | $\frac{Cx+D}{x^2+px+q}$ | 配方 → $\ln$ + $\arctan$ |

### 三角有理积分（$\int R(\sin x,\cos x)\,dx$）

| 类型 | 换元 | 化为 |
|---|---|---|
| $R(\sin x,\cos x)$ 通用 | $t=\tan\frac{x}{2}$（万能公式） | $\int R(t)\,dt$（有理函数） |
| $f(\sin x)\cos x$ | $t=\sin x$ | $\int f(t)\,dt$ |
| $f(\cos x)\sin x$ | $t=\cos x$ | $-\int f(t)\,dt$ |
| $f(\tan x)$ | $t=\tan x$ | $\int \frac{f(t)}{1+t^2}\,dt$ |

> **万能公式**是核武器但不优先——先试特殊换元（$\sin x$/$\cos x$/$\tan x$），万能公式最后兜底。

### 定积分的特殊技巧

| 技巧 | 适用场景 | 操作 |
|---|---|---|
| **对称区间 + 奇偶** | $\int_{-a}^a f(x)\,dx$ | 奇→0，偶→$2\int_0^a$ |
| **区间再现** | $\int_0^\pi$ 或 $\int_0^{2\pi}$ 含 $\sin/\cos$ | $x\to a+b-x$（Day 1 题 8.5 已练） |
| **华里士公式** | $\int_0^{\pi/2}\sin^n x$ 或 $\cos^n x$ | 分 $n$ 奇偶套公式 |
| **分段积分** | 含绝对值 / 分段函数 | 找零点 → 分段 → 逐段积分 |

### 变限积分函数的求导（D₁+D₂₂+D₂₃）

| 形式 | 求导结果 |
|---|---|
| $\frac{d}{dx}\int_a^x f(t)\,dt$ | $f(x)$ |
| $\frac{d}{dx}\int_a^{g(x)} f(t)\,dt$ | $f(g(x))\cdot g'(x)$ |
| $\frac{d}{dx}\int_{h(x)}^{g(x)} f(t)\,dt$ | $f(g(x))g'(x)-f(h(x))h'(x)$ |

> Day 2 的题 7（$F(x)=\int_{x^2}^{x^2+1}f(t)\,dt$）正是第三种形式——FTC 直接展开。

### 分段函数的积分（D₁+D₂₃）

找分段点 → 按区间拆积分 → 逐段代入不同表达式。**含绝对值的积分是特例**（Day 2 题 8 已练：$|u|$ 按 $u=0$ 分段）。

### 几何法（D₁+D₂₃）

利用定积分的**几何意义**（面积、面积正负）直接判断积分值，不必硬算。Day 1 的题 8.5（$I_1, I_2$ 正负判断）就用了几何法——画图看面积大小关系。

### 第二类换元法——三种三角换元结论（RAG 例 9.2 提取，必背）

| 被积函数含 | 换元 | 结论 |
|---|---|---|
| $\sqrt{a^2-x^2}$ | $x=a\sin u$ | $\int\sqrt{a^2-x^2}\,dx=\frac{a^2}{2}\!\left(\arcsin\frac{x}{a}+\frac{x}{a}\sqrt{1-\frac{x^2}{a^2}}\right)+C$ |
| $\sqrt{a^2+x^2}$ | $x=a\tan t$ | $\int\sqrt{a^2+x^2}\,dx=\frac{x}{2}\sqrt{a^2+x^2}+\frac{a^2}{2}\ln\!\frac{x+\sqrt{a^2+x^2}}{a}+C$ |
| $\sqrt{x^2-a^2}$ | $x=a\sec t$ | $\int\sqrt{x^2-a^2}\,dx=\frac{x}{2}\sqrt{x^2-a^2}-\frac{a^2}{2}\ln\!\left|\frac{x+\sqrt{x^2-a^2}}{a}\right|+C$ |

> **记忆口诀**：$\sqrt{a^2-x^2}$→$\sin$（反正弦）；$\sqrt{a^2+x^2}$→$\tan$（正切）；$\sqrt{x^2-a^2}$→$\sec$（正割）。**结果都有 $\frac{x}{2}\sqrt{\dots}\pm\frac{a^2}{2}\ln(x+\sqrt{\dots})$**——只差符号和反三角/arctan 项。
>
> **特殊值速查**（例 9.2 附注）：
> - $\int_0^{1/2}\sqrt{1-x^2}\,dx=\frac{\pi}{12}+\frac{\sqrt3}{8}$
> - $\int_0^1\sqrt{1-x^2}\,dx=\frac{\pi}{4}$（四分之一圆面积）

### 有理函数的积分——凑微分 + 配方（RAG 例 9.1）

**例 9.1** $\int\frac{x}{x^2+2x+2}\,dx$：
$$=\frac12\int\frac{2x+2}{x^2+2x+2}dx-\int\frac{1}{(x+1)^2+1}dx=\frac12\ln(x^2+2x+2)-\arctan(x+1)+C$$

> **套路**：分子凑分母的导数 $(x^2+2x+2)'=2x+2$，剩余项配方后套 $\int\frac{du}{u^2+a^2}=\frac1a\arctan\frac{u}{a}$。

## 六、几何应用——常见曲线的面积/体积/弧长大观（18 讲第 10 讲）

> 18 讲第 10 讲核心：把考研常考的几种曲线的**面积、旋转体积、弧长**预先算好，考场直接套。

### 三大基本公式（必背）

| 几何量 | 直角坐标 | 参数方程 | 极坐标 |
|---|---|---|---|
| **面积** | $S=\int_a^b\|f(x)-g(x)\|dx$ | $S=\int_{\alpha}^{\beta}y(t)x'(t)\,dt$ | $S=\frac12\int_{\alpha}^{\beta}r^2(\theta)\,d\theta$ |
| **旋转体积**（绕 $x$ 轴） | $V=\pi\int_a^b f^2(x)\,dx$ | $V=\pi\int_{\alpha}^{\beta}y^2(t)x'(t)\,dt$ | — |
| **弧长**（仅数一数二） | $L=\int_a^b\sqrt{1+f'^2(x)}\,dx$ | $L=\int_{\alpha}^{\beta}\sqrt{x'^2+y'^2}\,dt$ | $L=\int_{\alpha}^{\beta}\sqrt{r^2+r'^2}\,d\theta$ |

### 常见曲线结论表（RAG 从 18 讲第 10 讲提取）

**极坐标曲线**（心形线 / 双纽线 / 玫瑰线）：

| 曲线方程 | 面积 | 绕轴体积 | 弧长 |
|---|---|---|---|
| $r=a(1\pm\cos\theta)$（心形线） | $\frac{3}{2}\pi a^2$ | $\frac{8}{3}\pi a^3$（绕极轴） | $8a$ |
| $r=a(1\pm\sin\theta)$（心形线·竖） | $\frac{3}{2}\pi a^2$ | — | $8a$ |
| $r^2=a^2\cos2\theta$（双纽线） | $a^2$ | $\frac{\sqrt2}{8}\pi^2a^3$（绕 $y$ 轴） | — |
| $r^2=a^2\sin2\theta$（双纽线·竖） | $a^2$ | $\frac{\pi^2}{4}a^3$（绕极轴） | — |

**参数方程曲线**（摆线 / 星形线 / 笛卡尔叶形线）：

| 曲线方程 | 面积 | 绕轴体积 | 弧长 |
|---|---|---|---|
| 摆线 $x=a(t-\sin t),\,y=a(1-\cos t)$ | $3\pi a^2$ | $5\pi^2a^3$（绕 $x$ 轴） | $8a$ |
| 星形线 $x=a\cos^3t,\,y=a\sin^3t$ | $\frac{3}{8}\pi a^2$ | $\frac{32\pi}{105}a^3$（绕 $x$ 轴） | $6a$ |
| 叶形线 $x^3+y^3-3axy=0$ | $\frac{3}{2}a^2$ | — | — |

> **考场策略**：这些结论是**查表型知识**——不用现场推导，但必须记住"哪种曲线对应什么形状 + 面积量级"。考题通常是"给出曲线方程，求某几何量"，你直接套表。

### 旋转体积的壳法 vs 圆盘法

| 方法 | 公式 | 适用 |
|---|---|---|
| **圆盘法**（绕 $x$ 轴） | $V=\pi\int_a^b f^2(x)\,dx$ | 被积函数容易表示为 $y=f(x)$ |
| **壳法**（绕 $y$ 轴） | $V=2\pi\int_a^b x\|f(x)\|\,dx$ | 绕 $y$ 轴但函数是 $y=f(x)$ 形式 |

> 选择标准：**绕谁转，谁的变量在积分号外**。绕 $x$ 轴用圆盘（$dx$ 在外），绕 $y$ 轴用壳法（$x$ 在外）。

### 识别信号

- 见"曲线所围面积" → 先判**坐标系**（直角/参数/极），再套面积公式。
- 见"绕某轴旋转体积" → 判**绕谁转**，选圆盘 or 壳法。
- 见"弧长" → **仅数一数二考**；判坐标系后套弧长公式。

## 七、物理应用——微元法建模（18 讲第 12 讲，RAG 提取）

> 仅数学一、数学二。核心是**微元法**：取 $[x, x+\Delta x]$ 上的微元，近似看成常量，写出 $\mathrm{d}W$（或 $\mathrm{d}P$、$\mathrm{d}F$），再积分。

| 物理量 | 微元 | 积分公式 | 关键 |
|---|---|---|---|
| **变力做功** | $\mathrm{d}W = F(x)\,dx$ | $W=\int_a^b F(x)\,dx$ | $F$ 是位置的函数 |
| **抽水做功** | $\mathrm{d}W = \rho g x \cdot S(x)\,dx$ | $W=\int_0^h \rho g x S(x)\,dx$ | $S(x)$ 是截面面积，$x$ 是水深 |
| **静水压力** | $\mathrm{d}F = \rho g x \cdot L(x)\,dx$ | $F=\int_a^b \rho g x L(x)\,dx$ | $L(x)$ 是宽度，$x$ 是深度 |
| **变速位移** | $\mathrm{d}s = v(t)\,dt$ | $s=\int_{t_1}^{t_2} v(t)\,dt$ | 速度积分 = 位移 |
| **引力** | $\mathrm{d}F = \frac{Gm\mu\,dx}{r^2}$ | $\int \frac{Gm\mu}{(a-x)^2}\,dx$ | 万有引力 + 微元 |

> **微元法三步**：①取微元 $[x,x+dx]$ → ②在微元上近似看成常量写出 $\mathrm{d}(\text{物理量})$ → ③积分。
