# 微分方程

## 核心定义

微分方程 是含有未知函数及其导数的方程，阶数 是最高阶导数的阶数。通解 是含独立任意常数的解族，特解 是满足初值条件的确定解。

常考一阶类型：可分离变量方程 形如 $\frac{dy}{dx} =$ **$f(x)g(y)$**，分离后两侧积分；一阶线性方程 形如 $y' + p(x)y = q(x)$，通解公式 $y = e^{-\int p\,dx}\left[\int q \cdot e^{\int p\,dx}\,dx + C\right]$；齐次方程 形如 $y' = \varphi\left(\frac{y}{x}\right)$，令 $u =$ **$\frac{y}{x}$** 化为可分离变量。

常考二阶类型：二阶常系数齐次线性方程 $y'' + py' + qy = 0$，通过 特征方程 $r^2 + pr + q = 0$ 求解。特征根三种情况：**不等实根** $r_1 \neq r_2$，通解 $y =$ **$C_1 e^{r_1 x} + C_2 e^{r_2 x}$**；**重根** $r_1 = r_2 = r$，通解 $y =$ **$(C_1 + C_2 x)e^{rx}$**；**复根** $r = \alpha \pm \beta i$，通解 $y =$ **$e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$**。

二阶常系数非齐次线性方程 $y'' + py' + qy = f(x)$ 的通解 = **齐次通解 + 一个特解**。特解设法规则：若 $f(x) = P_m(x)e^{\lambda x}$，设 $y^* = x^k Q_m(x)e^{\lambda x}$，其中 $k$ 为 $\lambda$ 是特征根的 **重数**（不是根则 $k=0$，单根 $k=1$，重根 $k=2$）。

积分因子法：一阶线性方程的积分因子 $\mu(x) =$ **$e^{\int p(x)\,dx}$**，方程两边乘以 $\mu$ 后左端可化为 $(\mu y)'$ 的形式。

## 关键细节 / 操作步骤

1. 第一步：识别 **方程类型**——可分离、一阶线性、齐次型、二阶常系数，不同类型解法不同。
2. 第二步：可分离变量方程：先将 $x$ 和 $y$ 分到等号两侧 **$\frac{dy}{g(y)} = f(x)\,dx$**，再两侧积分。
3. 第三步：一阶线性方程：直接套通解公式 $y = e^{-\int p\,dx}\left[\int q \cdot e^{\int p\,dx}\,dx + C\right]$，或用积分因子法。
4. 第四步：齐次型 $y' = \varphi(y/x)$：令 $u = y/x$，则 $y = xu$，$y' =$ **$u + x\frac{du}{dx}$**，化为可分离变量方程。
5. 第五步：二阶常系数齐次：先写 **特征方程 $r^2 + pr + q = 0$**，按根型写通解。
6. 第六步：二阶常系数非齐次：先求 **齐次通解**，再按 $f(x)$ 类型 **设定特解形式**，代入方程定系数。
7. 第七步：特解设法中的 $k$ 值——$\lambda$ 不是特征根则 $k =$ **$0$**，是单根则 $k =$ **$1$**，是重根则 $k =$ **$2$**。
8. 第八步：复根情况通解 $y = e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)$，其中 $\alpha =$ **$-\frac{p}{2}$**，$\beta =$ **$\frac{\sqrt{4q-p^2}}{2}$**。
9. 第九步：初值问题：先求通解，再将初值条件 $y(x_0) = y_0$、$y'(x_0) = y_0'$ 代入定常数。
10. 第十步：应用题常见模型：指数增长/衰减 $y' = ky$、振动方程 $y'' + \omega^2 y = 0$、混合问题。

> **⚠️ 易错辨析**
> **"齐次方程"和"齐次线性方程"不是同一概念。** 前者指 $y' = \varphi(y/x)$ 型（通过 $u=y/x$ 换元），后者指方程右端为零（$q(x) = 0$）。**可分离变量法积分时常数项不能漏**——每一侧积分都会产生常数，合并为一个即可。**非齐次方程的通解 = 齐次通解 + 特解**，不能只写特解。**重根与共振时特解要乘 $x^k$**——反例：$y'' - 2y' + y = e^x$ 中 $\lambda = 1$ 是特征方程的重根，$k = 2$，应设 $y^* = Ax^2 e^x$ 而非 $Ae^x$。**通解中常数的个数等于方程阶数**——一阶一个常数，二阶两个常数，初值条件的个数也必须匹配。

> **💡 技巧与口诀**
> 口诀：**先判类型再变形，一阶看分离线性齐次，二阶先特征再设特解**。应用场景：看到常系数二阶方程，第一反应写特征方程，不要先尝试积分。**特解设法的记忆**：非齐次项 $P_m(x)e^{\lambda x}$ 中，$\lambda$ 与特征根的关系决定乘 $x$ 的次数 $k$。**初值条件必须在通解求出后再代入**，不能在中间步骤代入。**齐次型 $y'=\varphi(y/x)$ 的识别**：方程中 $x$ 和 $y$ 总以 $y/x$ 的组合形式出现。

> **📝 真题闭环**
> 求微分方程 $y'' - 4y' + 3y = 0$ 满足 $y(0) = 6$、$y'(0) = 10$ 的特解。

**解题思路**：审题抓"二阶常系数齐次+初值"，切入点是 **特征方程求根再定常数**；方法选择为标准特征方程法；计算关键点：特征方程 $r^2 - 4r + 3 = 0 \Rightarrow (r-1)(r-3) = 0 \Rightarrow r_1 =$ **$1$**$，$r_2 =$ **$3$**$（不等实根）；通解 $y =$ **$C_1 e^x + C_2 e^{3x}$**；代入初值：$y(0) = C_1 + C_2 =$ **$6$**，$y'(0) = C_1 + 3C_2 =$ **$10$**，解得 $C_1 =$ **$4$**，$C_2 =$ **$2$**；易错防范是代入初值时忘记对通解求导。

答案：**$y = 4e^x + 2e^{3x}$**
