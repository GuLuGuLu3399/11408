# 心态日记

> 这是一个 **Type 文档**：它定义「心态日记」这个笔记类型，同时充当模板。
> 新建当天的心态日记时，复制下面「模板」段，把日期改成今天即可。

## 这个类型是干什么的

你的**情绪空间**，不是复习记录。每天一篇，记录心态和状态的起伏。

- 想怎么写就怎么写，包括纯情绪宣泄。
- 不连打卡、不带愧疚感——某天不想写就不写。
- 和学习内容解耦：不背「反哺复习」的 KPI，也不是冲刺教练的领地。
- 周报看学习推进，这个看人。

## 模板（新建时复制这一段，改日期）

~~~
---
type: 心态日记
date: 2026-07-01
mood: 
---

# 2026-07-01

> 你的情绪空间，不是复习记录。想怎么写就怎么写。
> 下面三个提示只是破冰——不想答就删掉或留空，纯倾诉也完全 OK。

## 今天什么状态？

## 什么事压着你？又是什么让你松了口气？

## 明天想怎么调整？
~~~

## 字段说明

- `type: 心态日记` — 固定，不要改。
- `date` — 当天日期，ISO 格式 `YYYY-MM-DD`，用于排序。
- `mood` — 可选。填一个词形容今天（焦虑 / 平静 / 疲惫 / 还行 / …），不填也没事。
- 文件名建议：`docs/每日心态/2026-07-01.md`，日期前缀天然排序，一天一篇。
